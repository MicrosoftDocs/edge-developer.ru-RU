<span data-ttu-id="7bcf4-131">Для обеспечения защиты от отслеживания действий в Интернете Microsoft Edge использует два действия по обеспечению безопасности для классифицированных инспекторов.</span><span class="sxs-lookup"><span data-stu-id="7bcf4-131">To provide protection from tracking actions on the web, Microsoft Edge takes two enforcement actions against classified trackers:</span></span>
