# Ошибки консоли и коды состояния  

Эта ссылка поможет вам интерпретировать сообщения об ошибках и состоянии из [консоли](../console.md)DevTools.  Используйте приведенную ниже таблицу для поиска кодов с помощью префикса \ (где "x" — число&ndash;от 0 до 9 цифр \):

| Префикс кода | Область | Описание |  
|:--- | :--- |:--- |  
| [CSP143xx](#csp-codes) | Политика безопасности контента | Ресурсы, блокируемые с помощью заголовка `Content-Security-Policy` HTTP.  |  
| [CSS31xx](#css-codes) | Таблицы стилей | Ошибки, связанные с *веб-форматом формата шрифта* \ (WOFF \) и *внедренным* шрифтом OpenType \ (EOT \), возникают проблемы с источником шрифта и сервера узла.  |  
| [HTML1112 — 1300](#html-codes) | HTML | Большинство из них относятся к устаревшим кодам, относящимся к концепциям Microsoft Internet Explorer, таким как *режим документа* и *представление совместимости*.  |  
| [HTML1400-1532](#html5-parser-warnings) | Анализ HTML5 | Предупреждения проверки, вызываемые парсером HTML5.  |  
| [HTTP](#http-codes) | Ошибки и состояние сервера | Коды, возвращаемые удаленными серверами в ответ на HTTP-запросы.  |  
| [SCRIPT10xx](#javascript-syntax-errors) | Синтаксические ошибки JavaScript | Происходит, когда структура одного из операторов JavaScript нарушает одно или несколько синтаксических правил.  |  
| [SCRIPT50xx](#javascript-run-time-errors) | Ошибки среды выполнения JavaScript | Возникает, когда ваш сценарий пытается выполнить действие, которое система не может выполнить.  |  
| [SEC71xx](#security-codes) | Безопасность | Отражает условия безопасности, принудительно применяемые Microsoft EDGE, такие как *смешанное содержимое* и *Защита от слежения*.  |  
| [SVG560x](#svg-codes) | Масштабируемая векторная графика |  Обширная SVG-Отладка в настоящее время не поддерживается, но некоторые сообщения консоли предоставляются для целей отладки.  |  
| WebGL11_xxx | WebGL | Сообщения об ошибках из контекста WebGL.  Просмотрите также [GLSL ошибки](https://msdn.microsoft.com/library/dn611835.aspx).  |  
| [XML5xxx](#xml-codes) | XML | Ошибки при анализе и проверке XML.  |
