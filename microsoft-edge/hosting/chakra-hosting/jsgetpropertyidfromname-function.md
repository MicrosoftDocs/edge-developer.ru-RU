# Функция JsGetPropertyIdFromName
Получает идентификатор свойства, связанный с именем.  
  
## Синтаксис  
  
```cpp  
STDAPI_(JsErrorCode) JsGetPropertyIdFromName(  
   _In_z_ const wchar_t *name,  
   _Out_ JsPropertyIdRef *propertyId  
);  
```  
  
#### Параметры  
 `name`  
 Имя идентификатора свойства для получения или создания. Имя может состоять только из цифр.  
  
 `propertyId`  
 Идентификатор свойства в этой среде выполнения для заданного имени.  
  
## Возвращенное значение  
 Код `JsNoError` , если операция завершилась успешно, код ошибки в противном случае.  
  
## Комментарии  
 Идентификаторы свойств зависят от контекста и не могут использоваться в разных контекстах.  
  
 Требуется активный контекст сценария.  
  
## Требования  
 **Верхний колонтитул:** jsrt. h
