# Функция JsGetOwnPropertySymbols
Возвращает список всех свойств символа в объекте.  
  
## Синтаксис  
  
```cpp  
STDAPI_(JsErrorCode) JsGetOwnPropertySymbols(  
   _In_ JsValueRef object,  
   _Out_ JsValueRef *propertySymbols  
);  
```  
  
#### Параметры  
 `object`  
 Объект, из которого нужно получить символы свойств.  
  
 `propertySymbols`  
 Массив символов свойств.  
  
## Возвращенное значение  
 Код `JsNoError` , если операция завершилась успешно, код ошибки в противном случае.  
  
## Комментарии  
 Требуется активный контекст сценария.  
  
 Этот API поддерживается только в режиме Microsoft Edge.  
  
## Требования  
 **Верхний колонтитул:** jsrt. h
