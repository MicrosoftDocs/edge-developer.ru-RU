# Функция JsCreateFunction
Создает новую функцию JavaScript.
  
## Синтаксис  
  
```cpp  
STDAPI_(JsErrorCode) JsCreateFunction(  
   _In_ JsNativeFunction nativeFunction,  
   _In_opt_ void *callbackState,  
   _Out_ JsValueRef *function  
);  
```  
  
#### Параметры  
 `nativeFunction`  
 Метод, который нужно вызвать при вызове функции.  
  
 `callbackState`  
 Определяемое пользователем состояние, которое будет передано обратно обратному вызову.  
  
 `function`  
 Новый объект Function.  
  
## Возвращенное значение  
 Результат звонка (если таковой имеется).  
  
## Комментарии  
 Требуется активный контекст сценария.  
  
## Требования  
 **Верхний колонтитул:** jsrt. h
