Сборка: Microsoft. Web. WebView2. Core. dll

Аргументы события для события ContentLoading.

## Краткий обзор

 Участников                        | Описания
--------------------------------|---------------------------------------------
[IsErrorPage](#iserrorpage) | Значение true, если загруженное содержимое является страницей ошибки.
