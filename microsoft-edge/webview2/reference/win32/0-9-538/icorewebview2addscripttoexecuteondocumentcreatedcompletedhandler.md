# интерфейс ICoreWebView2AddScriptToExecuteOnDocumentCreatedCompletedHandler 

```

  : public IUnknown
```

Вызывающий объект реализует этот интерфейс, чтобы получить результат метода AddScriptToExecuteOnDocumentCreated.
