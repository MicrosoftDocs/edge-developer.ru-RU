# Техническая документация по конфиденциальности Microsoft Edge  

Наше обеспечение безопасности в браузере – это предоставление вам возможности защиты, прозрачности, контроля и соблюдения требований.  Чтобы uphold обязательства по обеспечению прозрачности продуктов Майкрософт, группа Microsoft Edge предоставляет документ, в котором объясняется, как работают возможности и службы Microsoft EDGE и как они влияют на вашу конфиденциальность.  Цель группы Microsoft Edge — дать вам полное представление об использовании данных, управлять различными функциями и управлять собранными данными, чтобы получить сведения, необходимые вам для принятия необходимых вам решений о конфиденциальности.  

В некоторых разделах бумаги в Microsoft Team содержатся инструкции по переходу на страницу параметров Microsoft EDGE и других страниц.  Для обеспечения единообразия Microsoft Edge использует сокращенный формат во всей статье: должны отобразиться URL-адреса, которые начинаются с `edge://` например `edge://favorites` или `edge://settings/privacy` .  Чтобы перейти к страницам, введите полужирный текст прямо в адресную строку Microsoft Edge.  Страницы доступны только для просмотра в Microsoft Edge.  

В техническом описании для классической версии Microsoft EDGE, а части бумаги могут быть функции или возможности, недоступные для всех пользователей.  Кроме того, в этом техническом документе обсуждаются функции и услуги, которые уже есть в продукте, но могут быть изменены в будущем.  Минимизация сбора данных в Microsoft Practices, а это значит, что данные хранятся в течение минимального промежутка времени, но время хранения может отличаться в зависимости от используемой функции или службы и может изменяться с течением времени.  

## Адресная строка и предложения  

Адресная строка позволяет вводить URL-адреса сайтов и находить их в Интернете.  По умолчанию адресная строка обеспечивает поиск и предложения сайтов, используя введенные символы.  Вам должны отобразиться предложения из избранного, журнал браузера, предыдущие поисковые запросы и служба поиска по умолчанию.  

:::image type="complex" source="./media/whitepaper-media/address-bar.png" alt-text="Адресная строка" lightbox="./media/whitepaper-media/address-bar.png":::
   Адресная строка  
:::image-end:::  

Чтобы быстрее просматривать и искать, по мере ввода адреса в адресную строку вводимые символы отправляются в службу поиска по умолчанию для службы поиска, чтобы отправлять вам предлагаемые запросы поиска.  В адресной строке Категория будет классифицирована как URL-адрес, поиск или Неизвестная запись.  Информация, вместе с выбранным предложением, положением выделения и другими данными адресной строки, отправляется в службу поиска по умолчанию.  Если ваша служба поиска является Bing, идентификатор, уникальный для вашего браузера, отправляется с данными для понимания запроса поиска и сеанса запроса.  Другие Автоподбор идентификаторов служб отправляются в поисковую систему по умолчанию, чтобы выполнить поисковые предложения.  Ваш IP-адрес и файлы cookie отправляются в службу поиска по умолчанию, чтобы увеличить релевантность результатов поиска.  Сигнал отправляется в службу поиска по умолчанию, когда вы выбираете адресную строку, чтобы сообщить поставщику о готовности к предоставлению рекомендаций.  Введенные символы и запросы поиска не отправляются в корпорацию Майкрософт, если только служба поиска не является Bing.  Чтобы разрешить отправку данных в службу поиска по умолчанию, перейдите в раздел "службы" и `edge://settings/privacy` в разделе " **услуги** " выберите **адресную строку** и включите функцию **поиска и предложения сайтов с помощью параметра мои введенные символы** .  Если вы отключите этот параметр, ваши введенные символы больше не будут отправляться в службу поиска по умолчанию.  Ваши запросы поиска по-прежнему отправляются в вашу службу поиска по умолчанию для предоставления результатов поиска.  Если Microsoft Edge определяет ввод в адресную строку, которая может содержать конфиденциальные данные, такие как учетные данные для проверки подлинности, локальные имена файлов или URL-данные, которые обычно шифруются, он не отправляет введенный текст.  Если вы хотите, чтобы Microsoft Edge собирал данные диагностики о адресной строке, включая количество предложенных запросов для всех поставщиков услуг поиска, перейдите к разделу `edge://settings/privacy` **Персонализация веб-интерфейса** , чтобы улучшить работу с **веб-интерфейсом, разрешив Microsoft использовать журнал браузера в этой учетной записи для персонализации рекламы, поиска, новостей и других параметров Microsoft Services** .  

Нажатия клавиш и посещаемые веб-сайты хранятся локально на устройстве на каждом профиле.  Чтобы удалить данные, перейдите к разделу `edge://settings/clearBrowserData` Очистка окна **просмотра данных** выберите пункт **журнал браузера** и нажмите кнопку **Очистить сейчас** .  Если служба Bing является службой поиска по умолчанию и вы вошли в Bing, вы можете удалить Поиск с помощью [панели мониторинга Microsoft "Конфиденциальность"][MicrosoftAccountPrivacy].  Чтобы очистить историю браузера и удалить веб-сайты в качестве предложений в адресной строке, перейдите на вкладку `edge://history` **Очистка данных обзора**и нажмите кнопку Очистить данные.  В Windows 10 вы можете очистить данные, которые корпорация Майкрософт собирает из адресной строки и функций для поиска предложений, перейдя в раздел **Начало**  >  **настройки**  >  **Privacy**  >  **диагностики конфиденциальности & отзыв** и в разделе **Удаление диагностических данных** нажмите кнопку **Удалить**.  Все остальные данные удаляются после 36 месяцев.  

Если вы вошли в Microsoft Edge с помощью рабочей или учебной учетной записи Майкрософт, а также Microsoft Search, то анонимный токен, представляющий вашу учетную запись, отправляется с запросом на предоставление функций, связанных с учетной записью, например результатов, относящихся к вашей компании.  

Все данные передаются безопасно по протоколу HTTPS.  Если служба [Bing][|::ref1::|Main] является службой поиска по умолчанию, Поиск и введенные символы будут сохранены в течение 6 месяцев.  

Если вы ищете одно слово в поле адрес, Microsoft Edge может отправить одно слово на DNS-сервер, чтобы узнать, соответствует ли он узлу в вашей сети, и попытаться подключиться к соответствующему узлу.  Параметр позволяет перейти к определенному узлу вместо поиска.  Например, если ваш маршрутизатор использует имя узла `router` и вы вводите `router` адресную строку, вы можете перейти к нему и `https://router/` найти слово `router` с помощью службы поиска по умолчанию.  Эта функция не управляется **предложением "показать мне Поиск и сайты" с помощью параметра "показывать введенные символы",** так как она не предусматривает отправки данных в поисковую систему по умолчанию.  

Кроме того, чтобы включить или отключить отправку данных в службу поиска по умолчанию, перейдите на страницу " `edge://settings/search` **Показать Поиск и предложения сайтов" с помощью параметра мои введенные символы** .  Чтобы изменить поисковую систему по умолчанию, перейдите в `edge://settings/search` раскрывающееся меню **адресную строку и выберите средство поиска, используемое в** раскрывающемся списке.  Если вы используете режим InPrivate или гость, Автоподбор будет отключен.  В режиме InPrivate отображаются предложения из локального браузера, такие как журнал браузера или поиск в прошлом поиске, но вводимые символы не отправляются в поисковую систему по умолчанию.  В гостевом режиме не отображаются никакие предложения и не отправляются введенные символы в поисковую систему по умолчанию.  

Данные, собранные другими службами поиска, следуют политике конфиденциальности компании.  

## Им  

Автозаполнение в Microsoft Edge повышает производительность, позволяя сохранять пароли, сведения о платежах, адреса и другие данные ввода форм.  После посещения сайта и начала заполнения формы Microsoft Edge использует сведения о заполнении формы для сопоставления сохраненных данных автозаполнения с формой.  Microsoft Edge предложит данные ввода форм, ранее сохраненные при обнаружении похожих форм.  Пароли и данные кредитной карты сохраняются только с явно заданными разрешениями для каждого пароля и карты.  

Адреса и другие элементы формы сохраняются по умолчанию.  Чтобы отключить сохранение и заполнение адреса и других данных формы, перейдите к `edge://settings/addresses` параметру " **сохранить и заполнить адреса**" и отключите его.  

Чтобы запретить Microsoft Edge выдать запрос на сохранение пароля, перейдите в раздел `edge://settings/passwords` **предложение сохранить пароли** и отключите его.  Если вы не хотите, чтобы Microsoft Edge выйдет из существующих сохраненных паролей и хотите удалить сохраненные пароли, `edge://settings/passwords` выберите **сохраненные пароли**.  Чтобы удалить все данные автозаполнения, `edge://settings/clearBrowserData` выберите пункт **Автозаполнение данных формы**, выберите нужный диапазон времени и нажмите кнопку **Очистить сейчас**.  

Если вы включили синхронизацию для своего профиля, данные автозаполнения синхронизируются во всех версиях Microsoft EDGE, где вы вошли в систему с помощью одних и тех же учетных данных.  Когда синхронизация включена, все данные автозаполнения хранятся на зашифрованных серверах Майкрософт.  Данные автозаполнения, хранящиеся на серверах Microsoft, используются только в целях синхронизации.  Вы можете отключить синхронизацию данных автозаполнения `edge://settings/profiles/sync` и рядом с вашим профилем. Нажмите кнопку " **Отключить синхронизацию** ".  Если вы включили синхронизацию для автозаполнения, удаление данных автозаполнения с устройства, когда вы вошли в Microsoft EDGE, удаляет данные автозаполнения с других устройств, которые вы вошли в одну и ту же учетную запись.  

Если вы посещаете веб-страницу и отправляете форму, Microsoft Edge отправляет сведения о форме, например хэш-код для типов записей hostname и автозаполнения \ (например, поле 1 ищет адрес электронной почты, поле 2 — Поиск пароля и т. д.), в службе Microsoft Forms Fill.  Службе не отправляются данные, введенные пользователем, или идентификаторы пользователей.  Эти сведения помогают Microsoft Edge правильно определять формы на разных веб-страницах.  Данные используются для сопоставления сохраненных данных автозаполнения с формой.  

Если вы используете гостевой режим, автозаполнение недоступно и новые записи автозаполнения не добавляются.  При просмотре InPrivate Microsoft Edge включает элементы автозаполнения, но новые элементы автозаполнения не добавляются.  

## Передавать  

Функция CAST в Microsoft EDGE позволяет отображать мультимедиа на другом экране с помощью Google CAST.  Вы можете получить доступ к этой функции с помощью **параметров и других (...)**  >  **Дополнительные инструменты**  >  **Передать мультимедиа на устройство**.  Функция приведения использует расширение маршрутизатора мультимедиа, которое по умолчанию не включено в Microsoft Edge.  При первом использовании функции CAST Microsoft Edge предлагает разрешение на установку расширения маршрутизатора мультимедиа.  

Нажмите кнопку **перезапустить** , чтобы установить расширения маршрутизатора мультимедиа из веб-магазина Chrome.  Для обеспечения актуальности расширения маршрутизатора мультимедиа при запуске Microsoft EDGE и через регулярные промежутки времени Microsoft Edge отправляет запросы на обновление в веб-магазине Chrome, в котором есть базовые данные о вашей версии Microsoft Edge.  Google может собирать некоторые данные, связанные с расширением маршрутизатора мультимедиа.  Чтобы удалить расширение маршрутизатора мультимедийных файлов, перейдите к `edge://flags` **маршрутизатору Edge-** by-Demand и отключите его.  Это также приведет к остановке обновлений из веб-магазина Chrome.  Расширение скрыто и не отображается в списке **установленных расширений** .  Для списка **установленных расширений** перейдите к `edge://extensions` .  

## Коллекции  

Вы можете собирать сайты, текст и изображения в Интернете, а также упорядочивать их по коллекциям в Microsoft Edge.  Все данные коллекций хранятся локально на устройстве и организуются в соответствии с профилем Microsoft Edge.  Если включена синхронизация для коллекций, созданные вами коллекции, включая заметки и примечания, будут доступны во всех версиях Microsoft EDGE, которые вошли в систему.  

Каждые 24 часа Microsoft Edge загружает список поддерживаемых сайтов, для которых существуют специальные шаблоны извлечения сущностей.  Эти шаблоны специфичны для каждого веб-сайта.  Когда вы создаете новый элемент в коллекции, Microsoft Edge проверяет, является ли сайт, из которого создается новый элемент коллекции, в списке поддерживаемых сайтов.  Если сайт входит в список, Microsoft Edge проверяет службу извлечения сущностей для шаблона определенного сайта.  С запросом к службе не связан ни одного идентификатора пользователя.  Шаблон пытается определить имя, цену, рейтинг, основное изображение и другие данные о собираемом элементе.  Если сайт, с которого создается новый элемент коллекции, не находится на сайте Supported List, Microsoft EDGE не загрузит шаблон.  Шаблоны позволяют создавать все элементы коллекции локально на устройстве.  Данные о элементах коллекции не отправляются службе для создания коллекции.  

Чтобы удалить шаблоны, хранящиеся на устройстве, и очистить данные кэша, перейдите к разделу Очистка данных просмотра `edge://settings/privacy` рядом с пунктом **Очистить данные для просмотра** и **Choose what to clear** выберите нужный диапазон и тип данных, а затем нажмите кнопку **очистить** . **Clear browsing data**  Кроме того, чтобы удалить кэшированные данные, `edge://settings/clearBrowserData` выберите нужный диапазон и тип данных, а затем нажмите кнопку **clear Now** (удалить).  

Если вы включите параметр **Показывать предложения из Pinterest в коллекциях** , если вы хотите увидеть, что коллекции выполняют поиск Microsoft Bing с помощью заголовка для поиска нужных страниц темы Pinterest.  Microsoft EDGE не отправляет данные о коллекциях в Pinterest.  Чтобы удалить предложения и прекратить поиск для страниц с разделами Pinterest, перейдите к `edge://settings/privacy` параметрам **Показать предложения из Pinterest в коллекциях** и отключите их.  

Коллекция недоступна при просмотре в режиме InPrivate или в гостевой работе.  

## Стает  

Если включены дополнительные диагностические данные, в том числе отчеты о сбоях, корпорация Майкрософт собирает данные диагностики, когда Microsoft Edge завершает работу или встречает другие проблемы с надежностью.  Диагностические данные используются для диагностики и устранения проблем с надежностью Microsoft EDGE и другими продуктами и службами Майкрософт.  

:::image type="complex" source="./media/whitepaper-media/crashes2.png" alt-text="Стает" lightbox="./media/whitepaper-media/crashes2.png":::
   Стает  
:::image-end:::  

Собираемые диагностические данные находятся в виде аварийного дампа, который содержит состояние устройства и программного обеспечения, собранные на момент возникновения проблемы с надежностью Microsoft Edge.  Аварийный дамп включает сведения о том, что произошло в момент возникновения проблемы с надежностью.  Такие сведения, как веб-сайт, который вы посещаете во время сбоя или использование центрального процессора, могут быть включены в диагностические данные.  Диагностические данные хранятся локально на устройстве и отправляются в корпорацию Майкрософт с помощью зашифрованной ссылки, если включено создание отчетов о сбоях.  Каждый аварийный дамп имеет идентификатор, уникальный для вашего устройства, идентификатор, который можно перезадать, уникальный для браузера, а также дополнительные диагностические данные (например, URL-адрес, использование ЦП и использование сети), чтобы помочь определить проблему.  Дополнительные диагностические данные подключены к аварийному дампу, чтобы помочь в диагностике проблемы с надежностью, например понять количество устройств, которые сталкиваются с проблемой и серьезность.  

Аварийные дампы отправляются в корпорацию Майкрософт и хранятся на защищенных серверах Microsoft в течение 30 дней, а затем удаляются.  Чтобы запросить удаление диагностических данных на устройствах с Windows 10, перейдите в **раздел**  >  **Параметры**запуска  >  Диагностика**конфиденциальности**  >  **& отзыв** и в разделе **Удаление диагностических данных** нажмите кнопку **Удалить**.  Собранные сведения о сбоях, например количество типов сбоев, хранятся для отчетов и целей улучшения продукта.  

Чтобы очистить диагностические данные, хранящиеся локально в файловой системе устройства, перейдите к `edge://crashes` кнопке " **Очистить все** " и нажмите кнопку "удалить все".  

Чтобы отключить сбор диагностических данных в Windows 10, перейдите к разделу Параметры **запуска**  >  **Settings**  >  **Privacy**  >  **Диагностика конфиденциальности & отзыв**.  Для версий Microsoft EDGE на всех остальных платформах отключите и отключайте `edge://settings/privacy` **продукты Майкрософт, отправляя отчеты о сбоях и данные о том, как использовать параметр браузера** .  Функция сбора диагностических данных также может быть отключена для предприятий с помощью [групповых политик, которые управляются вашей организацией][DeployedgeEnterprisePrivacySettings].  

## Диагностические данные о том, как использовать браузер  

Корпорация Майкрософт использует диагностические данные для улучшения качества продуктов и услуг Майкрософт, поддержания качества программного обеспечения Майкрософт и обеспечения их актуальности, а также помогает лучше понять, как работают продукты Майкрософт.Каждый раз, когда группа Microsoft Edge собирает данные, решение проверяется в соответствии с нужным вариантом.  Microsoft считает, что сбор данных и методики не полагается.  Служба Microsoft Edge стремится собирать только нужные вам данные и хранит их только в течение всего времени, если это необходимо для улучшения продуктов и услуг Майкрософт.  

По мере использования функций и служб в Microsoft EDGE и других приложений, использующих веб-платформу Microsoft EDGE, диагностические данные о том, как вы используете эти функции, отправляются в корпорацию Майкрософт. Функции и службы, упомянутые в техническом описании сбор диагностических данных.

На всех остальных платформах для управления сбором диагностических данных Войдите в `edge://settings/privacy` систему и включите или выключите **продукты Майкрософт, отправив отчеты о сбоях и данные о том, как использовать параметр браузера** . ## Лицензии на управление цифровыми правами и мультимедиа  

Если на веб-сайте есть мультимедийное содержимое, защищенное с помощью управления цифровыми правами \ (DRM \), Microsoft Edge использует конвейер безопасного воспроизведения для обеспечения того, что содержимое не копируется или недоступно для неправильного доступа.  В рамках компонента Microsoft Edge может хранить данные, связанные с управлением цифровыми правами, на устройстве, в том числе уникальные идентификаторы и лицензии мультимедиа, а также может передавать уникальный идентификатор серверу лицензирования мультимедиа, указанному поставщиком содержимого.  При использовании веб-сайта Microsoft Edge извлекает информацию DRM, чтобы убедиться, что у вас есть разрешение на использование содержимого.  Данные помогают проверить доступ к защищенному содержимому и обеспечить бесперебойную работу с мультимедиа.  

Microsoft EDGE поддерживает управление цифровыми правами с помощью API зашифрованных расширений мультимедиа \ (API ОБНОВЛЕННОЙ) для сайтов HTML5.  API ОБНОВЛЕННОЙ позволяет веб-сайтам взаимодействовать с поставщиком DRM, именуемым модулем для расшифровки содержимого \ (МЧР \).  Различные системы DRM, например Widevine в Google или PlayReady компанией Microsoft, могут поддерживаться реализацией МЧР разработчика.  Поставщики содержимого могут выбрать поддержку одной или нескольких потенциальных систем DRM и использовать функцию API ОБНОВЛЕННОЙ, чтобы определить, какую систему DRM использовать для определенного клиента.  Дополнительные сведения о конфиденциальности ОБНОВЛЕННОЙ можно найти в разделе [Конфиденциальность расширений зашифрованных файлов мультимедиа][W3cEncryptedMediaPrivacy].  

Microsoft EDGE поддерживает только PlayReady DRM в Windows 10.  PlayReady — это реализация DRM, позволяющая передавать мультимедийные данные, такие как 4000 видео и Dolby Atmos Audio.  Microsoft Edge использует API платформы Windows Platform Media Foundation для поддержки PlayReady.  Для проверки доступа к защищенному содержимому Microsoft Edge использует операционную систему Windows 10, в которой используется уникальный идентификатор \ (ID), и сообщает идентификатор с помощью службы PlayReady.  Все данные ОБНОВЛЕННОЙ, МЧР и браузера для PlayReady, которые сохраняются на устройстве, хранятся и поддерживаются в Microsoft Edge.  Дополнительные сведения о PlayReady [можно найти в разделе Простая система End to End][PlayreadyOverviewSimpleEndSystem].  

Microsoft EDGE поддерживает Widevine по технологии Google DRM, и этот параметр включен по умолчанию.  Microsoft Edge периодически извлекает обновления для Widevine с серверов Google.  Использование Widevine может включать в себя связь с Google.  Пользователи могут отказаться от использования Widevine в Microsoft EDGE, отключив флаг Widevine DRM в `edge://flags/#edge-widevine-drm` .  Widevine имеет возможность создать уникальный идентификатор устройства и передать его в Google.  Более подробную информацию о Widevine и конфиденциальности можно найти в разделе Политика конфиденциальности Google.  

Microsoft EDGE поддерживает управление цифровыми правами флэш-памяти по Adobe, которая используется на некоторых сайтах вместо HTML5.  Вы должны предоставить разрешение на отправку Adobe Flash, когда он запрашивается на сайте.  Если на сайте используется Adobe, Microsoft EDGE обеспечивает Adobe Access для доступа к уникальному идентификатору устройства.  Чтобы очистить и сбросить локально хранимые экземпляры идентификатора, перейдите к разделу `edge://settings/privacy` **Очистка данных обзора** выберите **команду выберите нужные**параметры, установите флажок для **файлов cookie и другие данные сайта**и нажмите кнопку **очистить** , чтобы удалить все сохраненные идентификаторы.  Чтобы остановить использование Adobe Flash DRM, перейдите на `edge://settings/content/flash` .  

Если вы запрашиваете доступ к зашифрованному носителю HTML5, например видеоролику, Microsoft Edge создает запрос лицензии для расшифровки мультимедиа.  МЧР, который вы используете, создает запрос лицензии, содержащий идентификатор запроса.  Запрос отправляется на сервер лицензирования.  Ни одна из частей запроса лицензии не содержит каких бы то ни было личных сведений, а требование к лицензии не хранится на устройстве.  

При возврате лицензии на мультимедиа создается идентификатор носителя, уникальный для пользователя и сайта.  Этот идентификатор не является общим для разных сайтов и для каждого сайта.  Идентификатор сеанса, используемый для идентификации сеанса воспроизведения, отправляется с помощью идентификатора носителя для расшифровки мультимедиа.  Идентификатор мультимедиа хранится на устройстве локально и может храниться вместе с поставщиком содержимого.  

Чтобы отключить все защиты от управления цифровыми правами и содержимым, перейдите на `edge://settings/content/protectedContent` **сайт, чтобы разрешить воспроизводить защищенное содержимое (рекомендуется)** и **Разрешить использование защищенного содержимого (может потребоваться перезапуск компьютера)** .  

*   Если отключить параметр **Разрешить воспроизводить защищенный контент для веб-сайтов** , это не приведет к воспроизведению воспроизведения для систем DRM на базе МЧР, таких как PlayReady и Widevine, но не для таких систем, как Flash Access DRM.  Для управления разрешениями для веб-сайтов Flash перейдите на страницу `edge://settings/content/flash` .  Отключение параметра приводит к тому, что функции мультимедиа перестают работать должным образом.  
*   Отключение параметра " **разрешить идентификаторы для защищенного содержимого** " предотвращает создание идентификаторов для системы управления правами для Flash Access и предотвращает периодическое получение обновлений Widevine из Google.  Если отключить этот параметр, это может привести к тому, что некоторые возможности мультимедиа на некоторых сайтах будут перестать работать должным образом.  

## Do Not Track  

Чтобы включить функцию "не **отслеживать** " в Microsoft EDGE, перейдите к `edge://settings/privacy` параметру **запросов "не отслеживать** " и включите его.  Если включить функцию "не **отслеживать** ", Microsoft Edge ОТПРАВЛЯЕТ заголовок DNT: 1 HTTP с запросами на трафик просмотра исходящих HTTP, HTTPS и SPDY на веб-сайты, на которые вы посещаете, чтобы запросить у каждого из них не использование средств отслеживания.  Однако настройка **запросов "не отслеживать"** не гарантирует, что веб-сайты не смогут отслеживать вас.  Некоторые сайты могут соблюдать запрос, отображая ваши объявления, не основанные на предыдущих браузерах.  Microsoft EDGE не управляет требованием запроса.  Чтобы предотвратить отслеживание веб-сайтов, перейдите к `edge://settings/privacy` параметру " **Защита от слежения** " и измените его на " **сбалансированный** " или " **ограниченный**".  

При использовании гостевого режима Microsoft EDGE не отправляет запросы **Do not Tracking** .  Если вы используете функцию просмотра InPrivate, Microsoft Edge отправляет запросы **Do Not Track** , если для профиля, который вы используете, включена функция **отправки запросов "не отслеживать"** .  

## Загрузки  

Microsoft EDGE позволяет безопасно и безопасно загружать файлы.  Вы можете выбрать, где на устройстве можно загрузить файлы `edge://settings/downloads` .  Если функция SmartScreen включена, сведения о файле, такие как имя файла и URL-адрес, отправляются в SmartScreen для проверки репутации файла.  Проверка репутации помогает избежать случайной загрузки известных вредоносных программ, которые известны на вашем устройстве.  Вы можете включить или отключить функцию SmartScreen `edge://settings/privacy` .  Дополнительные сведения о фильтре SmartScreen можно найти в разделе [SmartScreen](#smartscreen) .  

Чтобы просмотреть историю предыдущих загруженных файлов, перейдите на веб – `edge://downloads` .  Чтобы очистить данные браузера и удалить историю скачивания, перейдите на веб – `edge://settings/clearBrowserData` .  Удаление журнала скачивания из Microsoft EDGE не приводит к удалению файлов с устройства.  Удаление скачанных файлов с устройства не приводит к удалению файлов из журнала загрузки.  Если вы используете режим просмотра InPrivate или гость, история скачивания из сеанса удаляется, когда вы закрываете незакрытые или гостевые окна, но файлы сохраняются на устройстве.  

## Надстройки расширений и Microsoft Edge  

Вы можете установить расширения в Microsoft EDGE, чтобы добавить функциональные возможности в браузер.  При установке расширения с веб-сайта надстроек Microsoft EDGE или другого хранилища расширений Корпорация Майкрософт собирает сведения о расширении, чтобы помочь разработчикам и Microsoft понять, как используется расширение.  Microsoft Edge собирает Объединенные данные, в том числе количество загруженных расширений и сведения о том, как они выполняются, например данные о сбоях.  Корпорация Майкрософт использует объединенные данные с разработчиками расширения.  Комментарии и обзоры пользователей являются общедоступными на веб-сайте надстроек, а также совместно с разработчиками. Если вы вошли в Microsoft EDGE, установленные расширения с веб-сайта надстроек Microsoft Edge будут связаны с вашей учетной записью, чтобы предоставить рекомендации по расширению.  Данные используются в статистической обработке для понимания популярности расширений.   

Если вы хотите, чтобы расширения и параметры для ваших расширений были синхронизированы со всеми вашими версиями Microsoft EDGE, синхронизированными с вашей подпиской, перейдите к `edge://settings/profiles/sync` кнопке **включить синхронизацию** .   

Установка расширений необязательна и удаление любого расширения в любой момент можно перейти к следующему разделу `edge://extensions` .  Когда вы устанавливаете расширение, оно указывает, какие данные пользователей ему необходимы, и Microsoft Edge перед установкой расширения запрашивает разрешение на доступ.  Убедитесь, что расширение надежных источниках и безопасно перед установкой, и проверьте политику конфиденциальности разработчика для определенного расширения.  

Расширения обновляются с помощью службы обновления Microsoft Edge.  Microsoft Edge отправляет список установленных расширений службе обновления, чтобы проверить, доступно ли обновление.  Если вы установили расширение из веб-магазина Chrome, запросы отправляются в веб-магазин Chrome через регулярные интервалы времени для проверки наличия обновлений расширений.  Идентификатор расширения, версия расширения и сведения о Microsoft Edge включены в запрос на поиск обновлений.  Чтобы остановить запросы в веб-магазине Chrome, перейдите в раздел `edge://extensions` Удаление других расширений и выберите пункт **из других источников** .  

При импорте расширений из других браузеров, например Google Chrome, если расширение доступно на веб-сайте надстроек Microsoft EDGE, то Microsoft EDGE автоматически устанавливает расширение с веб-сайта надстроек Microsoft EDGE и включает его, если ранее вы включили расширение.  Если расширение недоступно на веб-сайте надстроек Microsoft EDGE, Microsoft Edge локально копирует и устанавливает свое расширение из Google Chrome, не открывая его или не подключаясь к веб-хранилищу Chrome.  Microsoft Edge запрашивает разрешение на включение расширения и разрешение расширений из других магазинов.  Если вы предоставили разрешение, Microsoft EDGE позволяет устанавливать расширения из других магазинов и обновлений в расширениях с помощью веб-магазина Chrome.  Чтобы включить или отключить этот параметр, перейдите на вкладку `edge://extensions` \ (или отключите \) **Разрешить расширения из других магазинов** .

## Семейная безопасность  

Корпорация Майкрософт предложит вам ряд средств для поддержания связи между семейством и обеспечьте надежную работу детей на устройствах с Windows 10, Xbox и Android с помощью средства запуска Microsoft.  

В группе семьи есть Семейные настройки, которые должны быть включены для детей при использовании Microsoft Edge.  Организатор групп семьи должен включить параметры для пользователей в группе.  Три основные функции, предлагаемые в семейную группу, — это веб-фильтрация, отчеты о действиях и безопасный поиск.  

Веб-фильтрация защищает детей в группе "семья" от перехода на ненужные веб-сайты или веб-сайты, специально заблокированные организатором семьи.  

Сведения о записях для веб-сайтов, которые вы посещаете, а также поиск, время экрана, используемые устройства и попытки посещения заблокированных сайтов.  Организатор групп семьи может видеть информацию по адресу [Family.Microsoft.com][MicrosoftAccountFamilyMain].  Данные собираются, шифруются в пути, отправляются в корпорацию Майкрософт и хранятся на защищенных серверах Microsoft Storage.  Данные собираются с помощью учетной записи ребенка дочернего приложения, чтобы они могли правильно управлять.  Отчеты об активности хранятся в [Family.Microsoft.com][MicrosoftAccountFamilyMain] в течение 30 дней, а затем удаляются после.  

Безопасный поиск. добавляет ключевое слово Safe к запросу заголовков в поисковых машинах.  Bing считывает безопасное ключевое слово и фильтрует результаты поиска, возвращенные дочернему элементу.  Другие поисковые системы могут возвращать отфильтрованные результаты, так как это ключевое слово тоже.  Все запросы ребенка собираются и становятся доступными для организатора семьи в отчетах об активности или на [Family.Microsoft.com][MicrosoftAccountFamilyMain].  Данные собираются с помощью учетной записи ребенка дочернего приложения, чтобы они могли правильно управлять.  

Данные браузера ребенка хранятся на серверах Microsoft и становятся доступными для родителей в течение 30 дней, а затем сразу же удаляются.  Данные могут быть удалены в любой момент на [информационной панели Microsoft о конфиденциальности][MicrosoftAccountPrivacy].  Чтобы отменить Просмотр данных, сохраненных локально на устройстве, перейдите в `edge://settings/clearBrowserData` .  

Дочерний элемент должен быть подписан на Windows 10 с помощью учетной записи Майкрософт, а параметры отчетов о действиях должны быть включены организатором семьи, чтобы включить сбор данных и предоставление общего использования организатором групп семьи.  Для сбора данных ребенок не обязан войти в Microsoft Edge.  Если функции семейной безопасности недоступны в вашей версии Windows 10, вы можете установить последнюю версию Windows, чтобы получить доступ к этим функциям.  

Режим гостя и просмотр InPrivate отключаются, если включена веб-фильтрация или отчеты о действиях.  

Организатор групп семьи может прервать сбор данных на портале семейной безопасности.  Дополнительные сведения о функциях семейной безопасности Microsoft можно найти [в разделе что такое группа семьи учетных записей Майкрософт?][MicrosoftSupport12413]  

## Отзыв  

Группа Microsoft Edge всегда прослушивает пользователей и дает отзыв.  Чтобы оставить отзыв в Microsoft EDGE, выберите **Параметры и другие**  >  **Справочные материалы и отзывы**  >  **Send feedback**.  Для последовательного веб-приложения \ (PWAs \) нажмите кнопку **Параметры и выберите пункт Дополнительно...**  >  **Отправьте отзыв в корпорацию Майкрософт**.  Вы должны предоставить сведения о обратной связи, но все прочие сведения являются необязательными.  Если в профиле Microsoft Edge обнаружено сообщение электронной почты, оно предварительно заполнено вместе с текущим URL-адресом сайта, который вы используете, и соответствующими диагностическими данными.  Диагностические данные могут содержать данные о включенных функциях Microsoft EDGE и использовании браузера.  Вы также можете дополнительно включить снимок экрана, файлы с устройства и запись вашего браузера.  Необязательные снимки экрана, файлы или записи могут содержать личные данные.  Данные используются только для целей диагностики и улучшения продукта.  

Отзывы пользователей безопасно отправляются в корпорацию Майкрософт с помощью HTTPS и хранятся на защищенных серверах Майкрософт.  Если вы включили в параметры конфиденциальности Microsoft Edge адрес электронной почты и **улучшите продукты Майкрософт, отправив отчеты о сбоях, а также сведения о том, как использовать параметр браузера** , на устройстве будет содержаться идентификатор, уникальный для установленного вами браузера.  Все диагностические данные, в том числе журналы диагностики, записи и вложения, хранятся в течение 30 дней.  Оставшиеся данные отзыва, в том числе необязательный снимок экрана, хранятся в течение 15 месяцев.  Если вы указали сообщение электронной почты с элементом отзыва, сделайте [запрос][MicrosoftConcernPrivacy] на удаление отзыва.  

## Геолокация  

Microsoft EDGE поддерживает [API географического положения][W3cGeolocationApiMain], который позволяет веб-сайтам получать доступ к данным о расположении с вашим разрешением.  Веб-сайты могут запрашивать Ваше местонахождение, например, при попытке найти ближайший магазин кофе поблизости.  Microsoft Edge всегда требует вашего разрешения, прежде чем предоставлять веб-сайтам доступ к вашему местоположению.  Чтобы управлять разрешениями или всегда блокировать доступ к местоположению сайтов, перейдите на сайт `edge://settings/content/location` .  

В правой части адресной строки Microsoft Edge указывает на то, что расположение не является общим.  

:::image type="complex" source="./media/whitepaper-media/geolocation2.png" alt-text="Location" lightbox="./media/whitepaper-media/geolocation2.png":::
   Location  
:::image-end:::  

Если вы разрешите общий доступ к расположению на сайте, Microsoft Edge отправляет сведения о локальной сети, такие как ваш IP-адрес и проходящую точку доступа Wi-Fi, к службе расположения Microsoft.  Служба Microsoft использует эти сведения для оценки ваших координат географического положения.  Оценка географического положения предоставляется на сайте, с которым вы согласились поделиться своим местоположением.  Чтобы указать, что Microsoft Edge предоставляет запрашивающему сайту более точное расположение в Windows 10, перейдите в раздел **Параметры**  >  **конфиденциальности**  >  **Location** и включите **разрешение на доступ к расположению на этом устройстве** и **разрешите приложениям доступ** к параметрам вашего местоположения.  Если вы отключите параметр **Разрешить доступ к расположению на этом устройстве** и **разрешите приложениям получать доступ** к параметрам вашего расположения, Microsoft Edge предоставляет примерное расположение для запрашивающего сайта.  Эти сведения доступны только в том случае, если вы уже разрешены для просмотра вашего местоположения.  Дополнительные сведения о параметрах расположения Windows можно найти в разделе [Служба расположения и конфиденциальность Windows 10][MicrosoftSupport4468240].  

Новый сгенерированный случайным образом идентификатор используется при создании запросов к службе расположения Microsoft Edge.  Служба расположения Microsoft EDGE не хранит ваши координаты географического положения в течение некоторого периода времени.  

Режим просмотра InPrivate использует разрешение на размещение для профиля, из которого был запущен сеанс InPrivate.  Гостевой режим всегда запрашивает разрешение перед предоставлением места для сайта.  

## Импорт данных браузера  

Microsoft Edge предлагает интерактивный и простой интерфейс, когда вы запускаете браузер в первый раз.  В ходе работы вы можете импортировать данные браузера в Microsoft Edge из другого браузера.  Во время импорта вы можете либо сохранить импортированные данные, либо удалить их и начать новую.  Данные включают Избранное, журнал браузера, автозаполнение, данные, расширения, параметры и другие данные браузера.  Данные браузера из более ранних версий Microsoft EDGE автоматически импортируются при обновлении до новой версии Microsoft Edge.  С помощью подтверждения Microsoft Edge импортирует данные браузера из других браузеров, например Google Chrome, Mozilla Firefox или Internet Explorer, а также от того, с помощью какого браузера он импортируется в зависимости от используемого вами браузера.  Импорт данных выполняется локально на вашем устройстве, хранится локально и не отправляется в корпорацию Майкрософт, если только вы не согласны синхронизировать данные браузера.  

:::image type="complex" source="./media/whitepaper-media/migration.png" alt-text="Import" lightbox="./media/whitepaper-media/migration.png":::
   Import  
:::image-end:::  

При импорте расширений из другого браузера (например, Google Chrome) Microsoft Edge импортирует локальную копию и запрашивает разрешение до ее включения, если это расширение не доступно на веб-сайте надстроек Microsoft Edge.  Разрешения для некоторых расширений могли измениться.  Чтобы просмотреть разрешения на расширение, перейдите на `edge://extensions` .  

Вы можете в любое время импортировать данные из другого браузера `edge://settings/importData` .  

## Установка и обновление  

Вы можете скачать и установить Microsoft EDGE на таких платформах, как Windows и macOS.  Microsoft Edge использует службу обновления для обеспечения актуальности и защиты вашей версии Microsoft Edge.  

При загрузке и установке Microsoft Edge информация об устройстве, например канале выпуска, основные сведения о оборудовании, идентификаторы обновления, идентификатор, уникальный для вашего устройства, а также идентификатор, уникальный для вашего браузера, будут отправлены в корпорацию Майкрософт в процессе установки.  IP-адрес устройства отправляется в службу обновления, но Последнее десятичное значение будет очищено для дополнительной защиты конфиденциальности.  Во время каждого сеанса просмотра создается новый сгенерированный случайным образом маркер для установки обновленных версий Microsoft Edge.  Маркер не связан с личными данными и используется только для процесса установки и обновления и для улучшения службы обновления.  

Microsoft Edge проверяет процесс установки и обновления с помощью службы обновления Microsoft Edge.  Если установка или обновление завершилась сбоем и отчеты о сбоях будут включены, журнал создается и отправляется в корпорацию Майкрософт.  Дополнительные сведения об отправке отчетов о сбоях в корпорацию Майкрософт можно найти в разделе " [сбои](#crashes) ".  Корпорация Майкрософт собирает сведения о том, как вы загрузили Microsoft EDGE, успешно выполнена установка и любые ненужные установки, чтобы лучше понять успешность загрузки Microsoft Edge.  

Автоматические обновления включены по умолчанию для всех пользователей Microsoft Edge.  На всех платформах Microsoft Edge проверяет наличие обновлений при запуске и периодически запускается.  На устройствах MacOS также можно периодически проверять наличие обновлений для продуктов корпорации Майкрософт.  Для организаций доступны дополнительные элементы управления и конфигурации.  Дополнительные сведения о дополнительных элементах управления и конфигурациях можно найти в статьях [Обновление][DeployedgeUpdatePoliciesUpdate].  

## режим Internet Explorer;  

Microsoft EDGE представляет собой упрощенный интерфейс интеграции Internet Explorer \ (IE).  Microsoft EDGE поддерживает только IE 11, а режим IE доступен только в Windows.  Функция режима IE доступна для организаций через групповые политики.  Администратор решает открыть определенные сайты в режиме IE в Microsoft Edge.  

:::image type="complex" source="./media/whitepaper-media/ie-mode.png" alt-text="Режим IE" lightbox="./media/whitepaper-media/ie-mode.png":::
   Режим IE  
:::image-end:::  

Microsoft Edge загружает список сайтов из места, определенного администратором, с помощью политики и кэширует файл, который определяет, какие сайты должны быть открыты в режиме Internet Explorer.  В зависимости от параметров Windows или IE 11 Microsoft Edge собирает диагностические данные об использовании режима IE, например о том, какие сайты находятся в сети, данные о производительности, данные о надежности и данные об использовании компонентов.  В Windows 10 данные диагностики собираются в соответствии с параметрами диагностических данных Windows.  В Windows 8,1 сведения о веб-сайте собираются, если пользователь выбрал функцию "пролистывание вперед" или "Рекомендуемые сайты" в Internet Explorer.  Режим IE может не соответствовать тем же параметрам сбора данных в параметрах конфиденциальности Microsoft Edge.  

Если администратор включил функцию обнаружения корпоративного сайта, данные журнала браузера периодически собираются, чтобы помочь администраторам просматривать сайты, которые посещает пользователи, и проверять, все ли обновления системы продолжают поддерживать эти сайты.  Дополнительные сведения об обнаружении корпоративных сайтов в IE11 можно найти в разделе [сбор данных с помощью обнаружения корпоративных сайтов][InternetExplorer11DeployGuideCollectDataEnterpriseSiteDiscovery].  

Данные браузера Internet Explorer хранятся локально в Microsoft EDGE и Internet Explorer.  Чтобы очистить данные для просмотра, `edge://settings/privacy` выберите пункт **Очистить данные браузера**.  

## Ненавязчивая реклама  

Для более удобного просмотра в Microsoft Edge есть возможность блокировать загрузку рекламных объявлений на сайтах, которые отображают ненужные и ненужные рекламные объявления.  При включении блокировки объявлений Microsoft Edge периодически скачивает с серверов Майкрософт самый последний список сайтов, в котором отображаются ненужные и ненужные рекламные баннеры, а также локальное хранение на устройстве.  В запрос на скачивание не включаются идентификаторы пользователей.  Если вы посещаете сайт, который находится в списке, Microsoft Edge блокирует все баннеры на сайте, и вы увидите `Ads blocked` сообщение.  Чтобы разрешить рекламу для сайта, нужно `edge://settings/content/ads` изменить параметры.  Помимо загрузки списка сайтов с рекламой на предмет вмешательства, функция блокировки рекламных объявлений не отправляет дополнительные сведения в корпорацию Майкрософт или запрашивает дополнительные сведения от Майкрософт при просмотре веб-страниц.  

## Список переходов  

Список переходов в Microsoft EDGE позволяет легко находить последние закрытые сайты, щелкнув правой кнопкой мыши значок Microsoft EDGE на панели задач.  Последние три закрытые вкладки хранятся локально для каждого профиля.  Вы можете удалить сайты из списка переходов в Windows 10, щелкнув каждый из них правой кнопкой мыши.  Если вы не хотите, чтобы в списке переходов отображались последние закрытые вкладки, вы можете либо очистить данные браузера, либо изменить параметры, `edge://settings/privacy` чтобы очистить Просмотр данных при закрытии браузера.  При использовании окна InPrivate Microsoft EDGE не добавляет закрытые сведения о вкладке в список переходов.  При использовании гостевого режима список переходов недоступен.  Дополнительные сведения о том, как очистить данные браузера, можно найти [в разделе Просмотр и удаление журнала браузера в Microsoft Edge][MicrosoftSupport10607].  

## Сетевое время  

Microsoft Edge использует сетевую службу времени Microsoft для отслеживания времени из внешнего источника, например сервера времени.  Через случайные интервалы или, когда Microsoft Edge встречает просроченный сертификат SSL, Microsoft Edge может отправлять запросы в корпорацию Майкрософт, чтобы получить время из надежного источника.  Запросы выполняются чаще, если Microsoft Edge обнаруживает неточную работу системных часов.  Если пользователь изменяет время в операционной системе и вступает в противоречие с правильным часовым поясом, неточность системных часов возникает.  Служба сетевого времени Microsoft используется для получения координированного Всемирного времени (UTC).  Запросы не содержат cookie-файлы или идентификаторы пользователей, и никакие данные не записываются в журнал.

### Новости Майкрософт  

Для адаптации контента к взаимодействию и параметрам настройки страница создания вкладки в Microsoft Edge хранит файлы cookie со случайным созданием идентификаторов на устройстве.  Для подгонки содержимого к общему региону также используется очищенная версия IP-адреса.  Файлы cookie сохраняются на устройстве до тех пор, пока не будет очищено `edge://settings/siteData` .  

Чтобы запретить персонализацию рекламы, перейдите в раздел " [Параметры рекламы][MicrosoftAccountPrivacyAdSettings] [" на панели мониторинга "конфиденциальность][MicrosoftAccountPrivacy] " и отключите функцию **просмотра персонализированных баннеров в** параметрах браузера.  Вы также можете отключить плитки Экспресс-ссылки, нажав **кнопку "Настройка"**,  >  **Custom** и отключив параметр " **Показывать быстрые ссылки** ".  Microsoft Edge использует локальный журнал браузера для персонализации плиток быстрых ссылок, и вы можете удалить или создать новые плитки.  Данные хранятся только локально на устройстве, на каждом профиле.  

Поле поиска на странице "Новая вкладка" выполняет поиск Bing на основе введенных запросов поиска.  Для автоматического предоставления предложений и результатов поиска Microsoft Edge использует введенные символы, поисковый запрос, IP-адрес и идентификаторы поиска с помощью Bing.  Поле поиска можно настроить с помощью групповой политики для предоставления результатов поиска из Microsoft Search, возвращая данные из своей организации, такие как документы и содержимое интрасети.  Чтобы обеспечить интегрированный поиск, Microsoft Edge хранит файлы cookie локально на устройстве.  

Если вы вошли в Microsoft Edge с помощью учетной записи Майкрософт, вы можете управлять действиями браузера, связанными с новой страницей вкладки, на [панели мониторинга конфиденциальности Microsoft][MicrosoftAccountPrivacyAdSettings].  

Microsoft Edge собирает диагностические данные о том, как использовать страницу "Новая вкладка", например взаимодействие с полем поиска и щелкая элементы Экспресс-ссылки, если **улучшить продукты Майкрософт, отправив отчеты о сбоях, и данные о том, как использовать** параметр браузера `edge://settings/privacy` .  Браузер отправляет диагностические данные о том, как вы можете использовать страницу новостей Майкрософт в Майкрософт для понимания взаимодействия пользователя с новостями и улучшениями продуктов Майкрософт.  Вы можете отключить содержимое новостей Microsoft, нажав кнопку "настроить" на вкладке "создать".  Новостные данные отправляются в корпорацию Майкрософт по протоколу HTTPS и хранятся в течение 13 месяцев, после чего они объединены и сделаны анонимными.  

На странице Новая вкладка также можно задать фоновое изображение.  Изображение хранится локально на устройстве и может быть удалено путем удаления изображения или отправки нового изображения.  Сведения об этом изображении не передаются в корпорацию Майкрософт.

## On startup (Запуск)  

С помощью Microsoft EDGE Вы можете выбрать, где вы остановились, открыв последние открытые вкладки из предыдущего сеанса просмотра.  Если вы выберете параметр продолжить с того места, с **которого вы остановились** , в `edge://settings/onStartup` том числе из предыдущего сеанса, включая cookie-файлы сеанса, они будут доступны при запуске, чтобы восстановить вкладки из предыдущего сеанса, и сохранить вход на сайты, на которые вы посещаете.  Если вы очищаете данные для просмотра при каждом закрытии браузера, но выбрали параметр **продолжить в том же месте** , указанные данные будут удалены, но URL-адрес будет сохранен для следующего сеанса.  

Кроме того, вы можете настроить Microsoft Edge так, чтобы при запуске будут открываться определенные страницы.  Указанные вами страницы хранятся локально на устройстве и зависят от профиля.  Если вы включили синхронизацию для параметров в `edge://settings/profiles/sync` , указанные страницы синхронизируются во всех версиях Microsoft EDGE, где вы вошли в систему.  

Вкладки "частный режим" и "гость" не восстанавливаются при запуске.  

## Монитор паролей
Корпорация Microsoft Edge заботится о том, что вы безопасно в Интернете.  Если вы вошли в Microsoft EDGE, то для обеспечения конфиденциальности личных и безопасной личной информации монитор паролей предупреждает вас о том, что ваши учетные данные были предоставлены в случае нарушения данных стороннего поставщика.  Если включен монитор паролей, сохраненные учетные данные хэшируются и шифруются локально на устройстве, отправляются на серверы Microsoft по протоколу HTTPS и сравниваются с зашифрованным списком известных нарушений учетных данных.  Ваш идентификатор учетной записи для входа надежно отправляется вместе с хешированными и зашифрованными учетными данными в службе монитора паролей.  Если учетные данные находятся в списке известных нарушений, корпорация Майкрософт отправляет зашифрованный ответ обратно в вашу версию Microsoft EDGE, чтобы предупредить вас о том, что ваши учетные данные были обнаружены как часть атаки или нарушений.  После завершения проверки никакие данные не будут храниться на серверах Майкрософт.   

Эта функция доступна только для пользователей, которые вошли в Microsoft Edge.  Microsoft Edge запрашивает разрешение на включение монитора паролей.  Чтобы включить или отключить контроль паролей, перейдите на `edge://passwords` .

## Платежи  

Microsoft Edge помогает вам повысить продуктивность работы, позволяя сохранить платежную информацию в своем профиле браузера и предложить автоматически вводить в них данные, когда они понадобятся при просмотре.  Если вы сталкиваетесь с похожей формой оплаты, Microsoft Edge предложит заполнить форму с помощью сохраненной информации.  Кредитная карта и другие платежные данные сохраняются только с вашим явным разрешением.  

Microsoft Edge предлагает вам, если вы хотите сохранить платежную информацию, если включено автозаполнение платежа.  Информация шифруется локально на вашем устройстве.  Вы можете удалить все сохраненные платежные данные в `edge://settings/payments` .  После удаления сохраненных платежных данных данные больше не отображаются в виде предложения автозаполнения.  Вы можете отказаться от того, чтобы сохранить платежную информацию, выключив функцию в `edge://settings/payments` .  

Microsoft EDGE поддерживает API PaymentRequest, позволяя заплатить за покупки с помощью автозаполнения.  API PaymentRequest позволяет продавцу запрашивать следующие данные: номер кредитной карты, срок действия кредитной карты, полное имя, адрес выставления счета, адрес электронной почты, номер телефона и адрес доставки.  API сообщает продавцу о том, что у пользователя есть сведения о кредитной карте, но он не предоставляет никаких сведений у получателя, пока вы не согласны.  Вы можете отключить эту функцию в `edge://settings/privacy` .  

Если ранее вы сохраняли данные об оплате в учетную запись Майкрософт, она также доступна для автозаполнения в браузере.  Сведения о платежах, хранящиеся в вашей учетной записи Майкрософт, синхронизируются на разных устройствах.  Если вы ранее сделали любые покупки Xbox или Microsoft Store, возможно, у вас уже есть информация об оплате, сохраненная в учетной записи Майкрософт.  При заполнении платежа карточка из учетной записи Майкрософт маскируется и размещается только после двухфакторной проверки подлинности.  Маска обеспечивает дополнительную защиту при получении сведений о платежах.  

## Персонализация рекламы, поиска, новостей и других служб Майкрософт  

Если вы разрешаете личную настройку, группа Microsoft Edge собирает и использует историю обзора Microsoft Edge для персонализации работы и рекламы в [Bing][|::ref3::|Main], Microsoft News и других службах Майкрософт.  Персонализация обеспечивает более важные и полезные результаты поиска, рекламу и содержимое новостей.  Например, если Рабочая группа Microsoft Edge выводится в зависимости от того, что вы предпочитаете использовать для покупок в определенном магазине, то реклама, которую вы видите, может быть адаптирована для продуктов из определенного магазина.  Кроме того, если вы часто смотрите на блоги и прочтите статьи о командировках, ваш канал новостей может включать в себя более важные новости о поездках.  
 
Эта функция доступна только для пользователей с учетной записью Microsoft, отличной от ребенка.  Эта функция недоступна для пользователей, которые вошли в Microsoft Edge с помощью рабочей или учебной учетной записи.  

История браузера собирается и используется для персонализации только в том случае, если выполнены все четыре условия:  

*   Пользователь входит в учетную запись Майкрософт, которая не является дочерней. 
*   Пользователь предоставил разрешение на сбор и использование данных для персонализации.  
*   Групповые политики пользователя, управляемые Организацией \ (работодатель, учебное заведение и т. д.) разрешают личную настройку.  
*   Пользователь не использует браузер в гостевом режиме или InPrivate.  

Журнал браузера и другие важные данные передаются по протоколу HTTPS и присоединяются к данным учетной записи Майкрософт.  Журнал браузера хранится на защищенных серверах Майкрософт.  Вы можете просмотреть и удалить ранее опубликованный журнал браузера, перейдя на [панель мониторинга конфиденциальности Майкрософт][MicrosoftAccountPrivacy].  История браузера хранится на защищенных серверах Майкрософт в течение не более 45 дней.  После 45 дней данные удаляются и не используются для персонализации.  

Вы можете изменить свои интересы или отказаться от персонализированных рекламных объявлений из [параметров рекламы][MicrosoftAccountPrivacyAdSettings] на [панели мониторинга Microsoft "Конфиденциальность"][MicrosoftAccountPrivacy].  

> [!NOTE]
> Отключение персонализированных баннеров на [информационной панели Microsoft о конфиденциальности][MicrosoftAccountPrivacy] не выключает сбор и использование журнала браузера для персонализации результатов поиска и содержимого в вашем канале новостей.  Чтобы отключить сбор и использование журнала браузера Microsoft Edge для персональных результатов поиска и новостей, перейдите к разделу `edge://settings/privacy` **Персонализация веб-интерфейса** , чтобы улучшить работу с веб-интерфейсом, **разрешая Майкрософт использовать журнал браузера в учетной записи для персонализации рекламы, поиска, новостей и других параметров Microsoft Services** .  Если вы отмените общий доступ к данным, корпорация Майкрософт больше не собирает и не использует журнал браузера для персонализации баннеров, результатов поиска и новостей. Дополнительные сведения о персонализации в Microsoft EDGE можно найти в статьях [журнал браузера Microsoft Edge для персонализированной рекламы и][MicrosoftSupport4532583]работы.  

## Печать  

Microsoft EDGE позволяет печатать веб-страницы, PDF-файлы и другие данные с помощью различных устройств и приложений.  При печати на принтере, приложении или PDF-файле Microsoft Edge отправляет команды и данные файла в операционную систему вашего устройства.  Эти сведения не отправляются в корпорацию Майкрософт и все данные, передаваемые в операционную систему вашего устройства для печати, удаляются сразу после завершения или отмены печати.  Чтобы изменить место назначения печати, перейдите на `edge://settings/printing` .  

Кроме того, вы можете печатать веб-страницы и файлы в формате PDF с помощью Microsoft Print to PDF, при этом никакие данные о файле не будут переданы в корпорацию Майкрософт.  Все заметки, внесенные в PDF-файл, сохраняются локально в файле.  

## Профили  

Профили в Microsoft Edge позволяют разделять данные браузера на независимые профили.  Данные, связанные с одним профилем, отделены от данных, связанных с другими профилями.  Например, если вы настроили свои рабочие учетные записи в разных профилях, вы не синхронизируйте свое личное Избранное и историю.  

Тем не менее, пользователи смогут легко переключаться между существующими профилями в Microsoft EDGE без необходимости использовать пароли.  Пользователи, у которых есть доступ к одному и тому же устройству, могут создать еще один профиль в той же версии Microsoft EDGE без разрешения текущего владельца профиля.  При удалении профиля из параметров Microsoft Edge окончательно удаляются данные для определенного профиля, который хранится на устройстве, например журнал браузера, избранное, данные заполнения форм и пароли.  Данные, синхронизированные с вашей учетной записью, могут храниться в облаке Microsoft и могут быть удалены с [информационной панели Microsoft о конфиденциальности][MicrosoftAccountPrivacy].  

Гостевой режим — это временный экземпляр нового профиля.  Это позволяет просматривать на устройстве другого пользователя без изменения профиля, выполнившего вход.  Просмотр данных из гостевого режима, например избранного, журнала браузера, пароля и данных заполнения формы, не сохраняется после закрытия всех окон гостевого режима.  Загруженные файлы хранятся на устройстве, но история загрузок удаляется.  Гостевой режим позволяет просматривать веб-страницы без автоматического входа в другие сайты.  Microsoft EDGE не отправляет на веб-сайты сведения о том, что пользователь просматривает данные в гостевом режиме.  При использовании гостевого режима разрешения на сбор диагностических данных о том, как вы посещаете браузер и веб-сайты, берутся из профиля Microsoft EDGE, на основе которого был запущен гостевой режим сеанса.  Все данные браузера для определенного сеанса гостевого режима удаляются после того, как все гостевые окна будут закрыты.  

Просмотр InPrivate — это частный режим просмотра, в котором хранятся журнал браузера, журнал загрузки, файлы cookie, данные сайта и данные заполнения форм.  Microsoft Edge сохраняет Скачанные файлы, а также все новые элементы избранного, созданные при просмотре InPrivate.  По умолчанию при просмотре InPrivate Корпорация Майкрософт не собирает сведения о веб-сайтах, которые вы посещаете для целей улучшения продукта.  Ваше учебное заведение, Рабочая область или поставщик услуг Интернета по-прежнему может видеть ваши действия в Интернете.  Данные просмотра для определенного сеанса InPrivate удаляются после закрытия всех окон InPrivate.  При использовании клавиатуры редактора метода ввода Windows \ (IME) для ввода текста и рукописного ввода может потребоваться собрать данные, чтобы улучшить распознавание и возможности предложений.  Для прекращения рукописного ввода и ввода данных из Microsoft при использовании клавиатуры Windows IME в закрытом и обычном окне просмотра выберите параметр "Параметры **запуска**"  >  **Settings**  >  **Privacy**  >  **Рукописный ввод & ввода персонализации**.  Дополнительные сведения о просмотре InPrivate можно найти в статьях [Просмотр InPrivate в Microsoft Edge][MicrosoftSupport4533513].  
  

## Прочтите вслух  

Microsoft Edge — это чтение вслух, которое читает содержимое веб-страницы для пользователя.  Начало чтения вслух в контекстном меню на странице или в разделе " **Параметры" и "Дополнительно" (...)**   >  **Прочтите вслух**.  Прочтите вслух предлагается несколько голосов, которые можно использовать для чтения содержимого веб-страницы.  Если вы используете голоса, [установленный в Windows 10][OfficeSupport4c83a8d8748642f78e462b0fdf753130] , в разделе **время & язык** в параметрах Windows 10, все ранее использовавшиеся голоса будут храниться в локальном кэше и могут быть очищены `edge://settings/clearBrowserData` .  

Когда вы начнете читать вслух, Microsoft Edge использует [веб-интерфейс Speech API][GithubW3cIncubatorCommunityGroupSpeechApi].  В зависимости от выбранного голоса содержимое страницы преобразуется из текста в речь с помощью предоставленной платформой клиентской библиотеки \ (например, в соответствии с вашей операционной системой) или серверной библиотеки, на базе которой используются функции Microsoft.  Если содержимое преобразовано в речь с помощью клиентской библиотеки, никакие данные не отправляются на серверы Microsoft.  Если содержимое преобразовано в речь с помощью функций Microsoft Speech Services (как указано в слове "Online" в любом из названий голосовых сообщений \), текст, а также маркер, созданный случайным образом, отправляется в корпорацию Майкрософт.  После завершения преобразования служба возвращает текст, наносимый из звукового файла, на ваше устройство.  Во время передачи данных из вашего устройства в корпорацию Майкрософт и наоборот они будут зашифрованы.  Текст, который отправляется в корпорацию Майкрософт, и созданный звуковой файл удаляется сразу после преобразования. никакие другие данные о вашем веб-содержимом не хранятся в течение некоторого периода времени.  

## Выпуск новых функций  

Для улучшения Microsoft Edge группа разработчиков Microsoft Edge всегда изучает пользователей.  В рамках обучения некоторые пользователи могут столкнуться с новыми функциями, прежде чем они станут доступны всем.  Для включения новых функций для случайно выбранных пользователей Microsoft Edge регулярно отправляет необходимые сведения о операционной системе, канале, версии, стране или регионе и другие данные конфигурации устройства в службу настройки Microsoft Edge.  Данные отправляются с помощью идентификатора, который можно использовать для переустановки, уникального для браузера.  Данные передаются службе по протоколу HTTPS.  Данные используются для получения обновлений, чтобы включить новые функции, для обеспечения актуальности и правильной работы Microsoft EDGE, а также для улучшения продуктов и служб Microsoft.  Для организаций доступны дополнительные элементы управления и конфигурации.  Дополнительные сведения о дополнительных элементах управления и конфигурациях для организаций можно найти в разделе [Конфигурация и эксперименты Microsoft Edge][DeployedgeConfigurationExperiments].  

Пользователь не может отключить обновления веб-браузера, контролируемые или настроенные вашей организацией, но управлять тем, какие данные об использовании продуктов будут отправляться в корпорацию Майкрософт, `edge://settings/privacy` и в разделе **Справка по улучшению Microsoft Edge** измените параметры.  

Чтобы понять, как новые функциональные возможности влияют на Microsoft EDGE и службы Майкрософт, Microsoft Edge отправляет уникальный идентификатор, который можно получить в браузере, и тег функций, который кодирует новые возможности, доступные в Microsoft EDGE и службах Майкрософт.  Новые функции помогают вам создавать оптимальные интерфейсы и самые удобные браузеры для всех пользователей.  Тег функциональных возможностей не является уникальным для установленной версии Microsoft EDGE и является общим для всех экземпляров Microsoft EDGE, использующих тот же набор функций, которые будут включены.  Microsoft Edge отправляет сведения в службах Майкрософт по протоколу HTTPS.  Браузер не отправляет информацию при просмотре InPrivate или в режиме гостя.  Чтобы предотвратить отправку данных, отключите и отключайте `edge://settings/privacy` **продукты Майкрософт, отправив отчеты о сбоях и данные о том, как использовать параметр браузера** .  Дополнительные сведения о том, как сбросить идентификатор, уникальный для браузера, можно найти в [диагностических данных о том, как использовать раздел "браузер"](#diagnostic-data-about-how-you-use-the-browser) .  

## Устранение ошибок навигации  

Если Microsoft Edge обнаруживает таймауты SSL-соединения, ошибки сертификата или другие проблемы в сети, которые могут быть вызваны порталом пределами (например, сеть Wi-Fi в отеле или аэропортах), Microsoft Edge отправляет запросы на `http://edge.microsoft.com/captiveportal/generate_204` код ответа и проверяет его.  Если запрос перенаправляется на другой URL-адрес, Microsoft Edge открывает URL-адрес на новой вкладке, предполагая, что он является страницей входа.  Запросы на страницу обнаружения портала предела не записываются, запросы не регистрируются, а файлы cookie не отправляются и не сохраняются.  На платформах Windows Microsoft Edge использует службу портала перезаписи Windows.  В противном случае используется служба портала Microsoft Edge.  Чтобы отключить службу, перейдите к разделу `edge://settings/privacy` **Использование веб-службы для разрешения ошибок навигации** и отключите его.  

## Вход в учетную запись и идентификация  

Вход в Microsoft Edge предоставляет дополнительные функции, повышающие эффективность работы браузера.  Чтобы выполнить вход без проблем, при первом запуске Microsoft Edge она попытается определить свое удостоверение в операционной системе.  Если Microsoft Edge обнаруживает ваше удостоверение из операционной системы, но вы не хотите оставаться в Microsoft EDGE, войдите в систему `edge://settings/profiles` и выйдите из нее или удалите свой профиль.  Если Microsoft EDGE не обнаруживает ваше удостоверение из операционной системы, вы можете войти в Microsoft Edge с сайта `edge://settings/profiles` .  

Если в операционную систему добавляется новое удостоверение, а в настоящее время профиль Microsoft EDGE не имеет удостоверения, Microsoft Edge добавляет к профилю определенное удостоверение.  Если вы входите в Microsoft Edge с учетной записью Майкрософт либо рабочей или учебной учетной записью и не указали удостоверение на вашем профиле Windows 10, указанная учетная запись будет добавлена в профиль Windows 10, если только вы не добавите его в Windows 10 при входе.  

При первом запуске Microsoft EDGE или входе в браузер вы не начнете синхронизировать данные в вашем профиле.  

Вход в Microsoft Edge включает единый вход \ (вы автоматически входите на веб-сайты, такие как Bing \) и другие возможности с питанием, такие как синхронизация.  Если вы хотите войти в Microsoft EDGE, а не на другие сайты Microsoft, такие как [Bing][|::ref4::|Main], вы можете выйти из определенного сайта.  Microsoft Edge создает файл cookie для выхода, указывающий, что Microsoft EDGE не будет входить на конкретный сайт для будущего посещения.  Вы можете снова войти на конкретный сайт, указав имя пользователя и пароль, а также удалив файлы cookie `edge://settings/privacy` .  Дополнительные сведения об удалении данных обзора можно найти в статьях [Просмотр и удаление журнала браузера в Microsoft Edge][MicrosoftSupport10607].  

Чтобы запретить связь с Microsoft EDGE, удалите профиль Microsoft EDGE или выйти из Microsoft Edge.  Чтобы удалить с устройства все данные, связанные с вашим профилем Microsoft EDGE, необходимо удалить профиль Microsoft Edge.  Удаление всех данных не приводит к удалению ранее синхронизированных данных, связанных с удостоверением.  

Ваша учетная запись в Microsoft EDGE на macOS совместно с другими приложениями Майкрософт.  Общее удостоверение позволяет входить в приложение Microsoft, не требуя отдельно вводить свои учетные данные, если вы вошли в другое приложение Microsoft на устройстве.  В macOS вы не войдете в Microsoft EDGE автоматически, основываясь на вашем состоянии проверки подлинности в другом приложении Microsoft.  При попытке войти в Microsoft Edge она предложит использовать учетные данные из другого приложения Microsoft на устройстве для эффективного входа в Microsoft Edge.  Если вы вошли в учетную запись в Microsoft EDGE, то при попытке входа в другие приложения Майкрософт можно использовать учетные данные Microsoft EDGE, которые помогут вам войти в другое приложение Microsoft на устройстве, не требуя повторного ввода учетных данных.  

Вы не можете войти в Microsoft Edge при использовании гостевого режима или InPrivate.  

## Фильтр SmartScreen  

Функция SmartScreen предназначена для безопасного просмотра веб-страниц.  При переходе на веб-сайты или загрузке файлов функция SmartScreen проверяет репутацию URL-адреса или файла.  Если фильтр SmartScreen определяет, что веб-сайт или файл является вредоносным, он блокирует перейти на сайт или загрузить файл.  

:::image type="complex" source="./media/whitepaper-media/smart-screen.png" alt-text="Фильтр SmartScreen" lightbox="./media/whitepaper-media/smart-screen.png":::
   Фильтр SmartScreen  
:::image-end:::  

При просмотре веб-страниц функция SmartScreen классифицирует сайты и загружаемые файлы как наиболее распространенный трафик, опасный или неизвестный.  Наиболее распространенный трафик — это популярные сайты, для которых определен фильтр SmartScreen.  Если вы переходите на сайт, помеченный как опасное, фильтр SmartScreen немедленно блокирует доступ к сайту.  При переходе на неизвестный сайт фильтр SmartScreen проверяет репутацию, чтобы определить, нужно ли вам получать доступ к сайту.  

Функция SmartScreen использует три типа проверки репутации.  Сначала функция SmartScreen проверяет URL-адреса сайтов, которые вы посещаете, в соответствии с локальным списком, чтобы определить, является ли сайт частью самого интенсивного трафика или является известным опасным сайтом.  Когда вы посещаете сайт высшего трафика, фильтр SmartScreen не отправляет URL-адрес службе SmartScreen.  Если URL-адрес находится в локальном списке опасных сайтов, фильтр SmartScreen блокирует его, что предотвращает загрузку любой части вредоносного веб-содержимого.  Microsoft Edge периодически скачивает обновленный список топ-трафика и опасное сайты для устройства.  Второй тип проверки URL-адреса — это асинхронная проверка репутации для службы SmartScreen.  Функция SmartScreen выполняет проверку всех URL-адресов, которые не отнесены к высшему трафику.  Microsoft Edge передает URL-адрес, важные сведения о сайте, идентификатор, уникальный для вашего устройства, а также общие сведения о расположении для службы SmartScreen, чтобы определить безопасность сайта.  Сведения, предоставленные Microsoft EDGE, позволяют службе определять новые опасные сайты и своевременно получать последние угрозы безопасности.  Результаты проверки URL-адресов хранятся локально на устройстве и автоматически очищаются в конце сеанса браузера.  Все запросы к службе SmartScreen выполняются с помощью шифрования HTTPS.  

В третьих, фильтр SmartScreen проверяет Скачанные файлы, чтобы предотвратить ущерб для вашего устройства.  Функция SmartScreen выполняет проверку репутации двоичных файлов синхронно по мере завершения скачивания.  Microsoft Edge отправляет сведения о файле, такие как хэш файла, имя файла, загружаемый URI и идентификатор, уникальный для вашего устройства, чтобы выполнить проверку репутации.  Все запросы SmartScreen выполняются с шифрованием по протоколу HTTPS.  Служба SmartScreen отправляет обратно результат проверки, что позволяет файлу полную загрузку.  Результаты будут храниться локально на устройстве.  

Служба SmartScreen хранит данные о проверке репутации и создает базу данных известных вредоносных URL-адресов и файлов.  Данные хранятся на защищенных серверах Майкрософт и используются только для служб безопасности Майкрософт.  Данные никогда не используются для идентификации или назначения вам.  При очистке кэша браузера удаляются все хранящиеся на локальном компьютере URL-данные SmartScreen.  Очистка журнала скачивания удаляет все сохраненные данные SmartScreen, связанные с загрузкой файлов.  

Функция SmartScreen включена по умолчанию для Microsoft Edge.  Чтобы отключить фильтр SmartScreen, перейдите в раздел `edge://settings/privacy` службы и в разделе **Сервис** отключите параметр **SmartScreen защитника Майкрософт** .  Этот параметр одинаков для всех профилей, связанных с установкой Microsoft EDGE на вашем устройстве.  Параметр не синхронизируется на разных устройствах.  Этот параметр действует для просмотра в режиме InPrivate и для гостевого режима.  Если управление устройством осуществляется с помощью групповых политик, заданных вашей организацией, этот параметр будет отображаться в Microsoft Edge.  Чтобы просмотреть этот параметр, перейдите на `edge://settings/privacy` .  Дополнительные сведения об этом можно найти в разделе [SmartScreen: вопросы и ответы][MicrosoftSupport17443].  

Кроме того, функция SmartScreen проверяет URL-адреса загружаемых файлов, чтобы убедиться в том, что они классифицированы как потенциально нежелательные приложения.  Блокирование потенциально нежелательных приложений способствует более продуктивной, производительной и отличные Windows.  Этот параметр отключен по умолчанию и доступен только на устройствах с Windows 10.  Чтобы включить эту функцию, перейдите на вкладку `edge://settings/privacy` **блокировать потенциально ненужные приложения** и включите этот параметр.  Дополнительные сведения о том, как потенциально нежелательные приложения классифицируются по категориям, можно найти в разделе [потенциально нежелательное приложение (к)][WindowsSecurityThreatProtectionIntelligenceCriteriaPotentiallyUnwanted].  Дополнительные сведения о том, как настроить параметр, можно найти в разделе [Определение и блокирование потенциально нежелательных приложений][WindowsSecurityThreatProtectionWindowsDefender].  

## Проверки орфографии  

Microsoft Edge проверяет правописание при вводе текста в браузере.  Служба проверки орфографии завершается локально на устройстве; Microsoft EDGE не отправляет сведения о введенных данных в корпорацию Майкрософт для проверки правописания.  Чтобы отключить эту функцию, перейдите в раздел `edge://settings/languages` **Проверка орфографии** отключите параметр для каждого языка.  

При добавлении нового языка в Microsoft Edge браузер загружает словарь для нового языка на устройство с помощью HTTPS.  Словарь используется для встроенной службы проверки орфографии.  При удалении языка из параметров Microsoft Edge удаляется словарь с устройства.  Гостевой режим не использует настраиваемый словарь из профиля или добавленных языков.  Чтобы добавить или удалить слова в локальном словаре, перейдите к разделу `edge://settings/languages` **Проверка орфографии** , выберите команду **Добавить или удалить слова**.  

## Предложите похожие сайты  

Для устранения опечаток URL-адресов в адресной строке, которая приводит к ошибке веб-сайта, Microsoft Edge может рекомендовать исправленный URL-адрес.  При возникновении ошибки навигации веб-сайта Microsoft Edge отправляет домен веб-адреса в службу Microsoft, чтобы предложить исправленный URL-адрес.  Microsoft EDGE не включает идентификаторы или маркеры с доменом.  Если служба находит предложение, она возвращает предложенный URL-адрес.  Корпорация Майкрософт хранит неверный домен и предлагаемый домен, чтобы помочь вам улучшить службу.  Чтобы облегчить переход на нужные сайты, эта функция включена по умолчанию.  Чтобы отключить эту функцию, перейдите в раздел " `edge://settings/privacy` службы" и в разделе " **услуги** " отключите параметр " **предлагать похожие сайты", если веб-сайт не удается найти** .  

## Синхронизация  

Вход в Microsoft Edge с помощью учетной записи Майкрософт позволяет синхронизировать данные браузера для всех версий Microsoft EDGE, которые вошли в систему.  Вы можете синхронизировать Избранное, параметры, форму заполнения данных \ (адреса и многое другое), пароли, расширения и коллекции.  Вы должны предоставить разрешение на включение синхронизации в Microsoft EDGE, а также включить или отключить все синхронизированные типы данных по отдельности.  Избранное включает все вкладки, ранее заданные в устаревшей версии Microsoft EDGE, которые синхронизируются вместе с остальными избранными.  Удаленные или измененные элементы избранного и другие данные из одной из указанных версий Microsoft Edge синхронизируются со всеми другими версиями Microsoft EDGE, в которых включена синхронизация.  Можно управлять конфигурацией синхронизации `edge://settings/profiles/sync` .  

:::image type="complex" source="./media/whitepaper-media/sync.png" alt-text="Синхронизация" lightbox="./media/whitepaper-media/sync.png":::
   Синхронизация  
:::image-end:::  

Для синхронизации с помощью дополнительных параметров подключения к устройству и данных конфигурации, необходимых для обеспечения синхронизации, например имени устройства, элемента и модели, отправляется сообщение.  Данные могут быть удалены с [панели мониторинга Microsoft Device][MicrosoftAccountDevices].  Для управления синхронизированным избранным перейдите на `edge://favorites` .  Чтобы управлять всеми другими типами данных, перейдите на `edge://settings/profiles` .  

При передаче между браузером и серверами Майкрософт все синхронизированные данные шифруются по протоколу HTTPS.  Синхронизированные данные также хранятся в зашифрованном состоянии на серверах Майкрософт.  Конфиденциальные типы данных, такие как адреса и пароли, перешифруются на устройстве перед синхронизацией.  Если вы используете рабочую или учебную учетную запись, все типы данных перед синхронизацией будут зашифрованы с помощью Microsoft Information Protection.  Все прочие синхронизированные типы данных сохраняются до тех пор, пока вы не удалите их, учетная запись удаляется, а учетная запись становится неактивной.  Идентификатор учетной записи вкладывается во все синхронизированные данные, так как идентификатор необходим для синхронизации на нескольких устройствах. 

Данные браузера InPrivate и гостевого режима не синхронизируются с учетной записью Майкрософт.  Однако Избранное, созданное в сеансах с закрытым подключением, синхронизируются в рамках вашей версии Microsoft EDGE, для которой включена синхронизация.  

## Блокировка отслеживания  

Microsoft Edge разработан для обнаружения и блокирования известных инспекторов.  Пользователи могут выбрать один из трех уровней защиты от слежения: Basic, сбалансированный и Strict.  Для защиты конфиденциальности пользователей по умолчанию выбрано значение сбалансированный.  Microsoft Edge обнаруживает отслеживаемые данные перед загрузкой на странице с помощью списка известных отслеживаемых источников.  Этот список будет периодически загружаться на устройство при обновлении списка.  Количество заблокированных и названий отслеживаемых инспекторов хранится локально на устройстве в статистических целях.  Чтобы очистить данные, перейдите на `edge://settings/privacy/blockedTrackers` .  Обнаружение и блокирование отслеживаемых данных осуществляется локально на устройстве.  Чтобы отключить защиту от слежения, перейдите на `edge://settings/privacy` .  Дополнительные сведения об отслеживании проблем можно найти [в статье сведения о предотвращении отслеживания в Microsoft Edge][MicrosoftAccount4533959].  

Вы можете отключить обновление списка с помощью следующей групповой политики, чтобы [Включить обновления компонентов в Microsoft Edge][DeployedgePoliciesComponentupdatesenabled].

## Переводчик  

В Microsoft EDGE Вы можете просматривать веб-страницы и переводить их на выбранные языки.  Встроенная функция перевода использует на устройстве службу, которая выявляет случайные части веб-страницы для обнаружения исходного языка.  Если обнаруженный язык не является одним из языков по умолчанию, Microsoft Edge предложит перевести веб-страницу на отображаемый язык или на другой выбранный вами язык.  Microsoft EDGE не переводит веб-страницу без вашего разрешения.  Вы также можете перевести веб-страницу, щелкнув ее правой кнопкой мыши и выбрав команду **перевести**.  Чтобы перевести веб-страницу, Microsoft Edge отправляет содержимое страницы и случайно сформированный маркер службе перевода Microsoft Azure по безопасному HTTPS-соединению.  Содержимое страницы не сохраняется ни в каких целях.  Чтобы запретить Microsoft Edge предлагать перевод веб-страниц, перейдите к `edge://settings/languages` предложению и отключите его, **чтобы перевести страницы, не** включенные в ваш язык.  

## Веб-приложения и закрепленные сайты  

Microsoft EDGE позволяет установить веб-приложения, созданные разработчиками веб-сайтов, и закрепить избранные сайты.  

При закреплении сайта он добавляется на панель задач или закрепление.  Данные хранятся на вашем устройстве локально.  Для некоторых сайтов сведения о том, является ли сайт закрепленным, предоставлен общий доступ к сайту, поэтому сайт не может запрашивать закрепление.  Вы можете управлять закрепленными сайтами на панели задач или закрепления.  Закрепленные сайты открываются в окнах Microsoft EDGE и используют те же разрешения сайтов и параметры диагностических данных, что и в конкретной версии Microsoft Edge.  

## WebView

Веб-представления Microsoft Edge позволяют разработчикам приложений размещать в приложениях приложения на родном языке в Windows 10 и Windows 7.  Приложения, размещенные в Microsoft Edge WebView, могут отправлять диагностические данные, такие как использование веб-платформы Microsoft EDGE и сайтов, которые вы посещаете в приложении, в корпорацию Майкрософт.  Чтобы включить сбор диагностических данных, перейдите к разделу `edge://settings/privacy` **помощь по улучшению параметров Microsoft Edge** .  Чтобы отключить сбор диагностических данных для Microsoft EDGE в Windows 10, перейдите к разделу Параметры **запуска**  >  **Settings**  >  **Privacy**  >  **Диагностика конфиденциальности & отзыв**.  Чтобы отключить сбор диагностических данных для всех прочих платформ, перейдите `edge://settings/privacy` в обычном сеансе просмотра и отключите оба **продукта Майкрософт, отправив отчеты о сбоях** и проведите для них поиск и данные о том, как использовать браузер, и **Повысьте эффективность поиска и работы с продуктами Майкрософт, отправив сведения о веб-сайтах, которые вы посещаете, в параметрах Microsoft Edge** .  Приложения, размещенные в Microsoft Edge WebView, могут собирать другие данные, управляемые коллекцией данных разработчика и соответствующими политиками конфиденциальности.  

## Веб-сайты, на которые вы посещаете для улучшения продуктов и услуг  

Если вы решили улучшить поиск и другие продукты и службы Майкрософт, Microsoft Edge отправляет сведения о веб-сайтах, которые вы посещаете в Microsoft.  Диагностические данные используются в агрегатах для улучшения продуктов и служб Майкрософт для всех пользователей, а также не собираются и не сохраняются вместе с учетной записью Майкрософт.  Отправка сведений о сайтах, которые вы посещаете в Microsoft EDGE, помогает Майкрософт понять, как быстро сайты загружают и увеличивают релевантность результатов поиска.  

Диагностические данные включают URL-адрес страницы, которую вы посещаете, метрики веб-сайта, название страницы, способ доступа к странице, сведения о содержимом страницы и другие сведения о навигации по страницам.  Некоторые диагностические данные прикреплены к идентификатору, уникальному для вашего устройства.  В противном случае данные диагностики связаны с идентификатором, который можно использовать для переустановки, уникальным для вашего браузера.  Идентификаторы не содержат ваших личных сведений.  Группа Microsoft Edge учитывает конфиденциальность диагностических данных за счет ограничения доступа к данным и (или) очистки конфиденциальных сведений.  Чтобы сбросить идентификатор, уникальный для браузера на устройствах с Windows 10, перейдите к разделу Параметры **запуска**  >  **Settings**  >  **Privacy**  >  **Диагностика конфиденциальности & отзыв** и в разделе > **диагностические данные** изменить с **полной** на **простой**.  На других платформах для создания нового заменяемого идентификатора для браузера перейдите на страницу " `edge://settings/privacy` **Поиск и изменение продуктов Microsoft", отправив сведения о веб-сайтах, которые вы посещаете в Microsoft Edge** , **— в** **положение выкл**.  Функция сброса может отличаться для устройств, управляемых с помощью групповых политик, заданных вашей организацией.  

Для Microsoft EDGE в Windows 10 этот параметр определяется параметром диагностических данных Windows.  Чтобы просмотреть отраженные параметры в Microsoft EDGE, перейдите в `edge://settings/privacy` раздел **Конфиденциальность** и **службы**.  Чтобы изменить параметры Windows, выберите команду Параметры **запуска**  >  **Settings**  >  **Privacy**  >  **Диагностика конфиденциальности & отзыв**.  На всех остальных платформах для управления сбором диагностических данных `edge://settings/privacy` включите или отключите функцию " **Поиск и продукты Майкрософт", отправив сведения о веб-сайтах, которые вы посещаете в Microsoft Edge** .  Этот параметр одинаков для всех профилей, связанных с установкой Microsoft EDGE на вашем устройстве.  Параметр не синхронизируется на разных устройствах.  Если управление устройством осуществляется с помощью групповых политик, заданных вашей организацией, параметр будет отображаться в `edge://settings/privacy` .  

Диагностические данные шифруются при передаче и хранятся на защищенных серверах Майкрософт в виде необработанных данных в течение 30 дней.  После того как диагностические данные не будут идентифицированы и объединены, необработанные диагностические данные удаляются.  Неидентифицированные и агрегированные диагностические данные хранятся в течение 2 лет, чтобы продолжить совершенствовать продукты и службы Майкрософт.  Так как диагностические данные не собираются и не хранятся в учетной записи Майкрософт, они могут не просматриваться и удаляться из [информационной панели Microsoft о конфиденциальности][MicrosoftAccountPrivacy].  На устройствах с Windows 10 для запроса на удаление диагностических данных, связанных с устройством, перейдите на вкладку **Пуск**  >  **Settings**  >  Диагностика**о конфиденциальности**  >  **& отзыв** и в разделе **Удаление диагностических данных** нажмите кнопку **Удалить** .  На других платформах диагностические данные удаляются или полностью обрабатываются и становятся анонимными в течение 30 дней.  

Для просмотра диагностических данных, собранных в Windows 10, перейдите к разделу Параметры **запуска**  >  **Settings**  >  **Privacy**  >  **Диагностика конфиденциальности & отзыв** и выберите пункт **Просмотр диагностических данных**.  На других платформах для просмотра данных перейдите к `edge://data-viewer` .  На других платформах чтобы включить средство просмотра диагностических данных, перейдите на страницу `edge://settings/privacy` " **Поиск и использование продуктов Майкрософт", отправив сведения о веб-сайтах, которые вы посещаете, в параметрах Microsoft Edge** .  

Диагностические данные никогда не отправляются при просмотре InPrivate или в гостевом режиме.  

## Application Guard в Защитнике Windows;  

Application Guard в Защитнике Windows \ (WDAG \) — это функция, которая доступна для организаций.  Если включен Application Guard в Защитнике Windows, Microsoft Edge открывает недоверенные сайты в изолированном контейнере, чтобы защитить ресурсы Организации от вредоносных сайтов или фишинг-атак.  Эта функция включена только для групповых политик, управляемых вашей организацией, и доступна только в последних версиях Windows 10.  WDAG собирает данные диагностики по улучшению качества программного обеспечения для открытия ненадежных сайтов в изолированном контейнере, например время, необходимое для открытия нового окна Application Guard.  

С вашего разрешения WDAG также собирает сведения о том, как использовать браузер и сведения о посещаемых вами сайтах.  Чтобы отключить сбор диагностических данных для Microsoft EDGE в Windows 10, перейдите к разделу Параметры **запуска**  >  **Settings**  >  **Privacy**  >  **Диагностика конфиденциальности & отзыв**.  Чтобы отключить сбор диагностических данных для всех прочих платформ, перейдите `edge://settings/privacy` в обычном сеансе просмотра и отключите **продукты Майкрософт, отправив отчеты о сбоях и проведите поиск и данные о том, как использовать браузер** , и **Повысьте эффективность поиска и работы с продуктами Майкрософт, отправив сведения о веб-сайтах, которые вы посещаете, в параметрах Microsoft Edge** .  

## Windows Information Protection  

Защита данных Windows \ (НЗП) помогает предотвратить случайное утечка корпоративной информации.  Она доступна только для организаций с помощью групповых политик, которые управляются вашей организацией.  НЗП включена для сайтов, которые идентифицированы как корпоративные ресурсы.  Определите, какие сайты являются корпоративными ресурсами, с помощью значка "Управление" в адресной строке.  НЗП обеспечивает выполнение таких функций, как предотвращение копирования и вставки данных из браузера или загрузки определенных файлов на сайты за пределами Организации.

НЗП доступен только в выпусках Windows 10 из августа 2016 или более поздней версии.
