# Справочные материалы по событию временной шкалы   




В режиме событий временной шкалы отображаются все события, инициированные во время создания записи.  Чтобы узнать больше о каждом типе событий временной шкалы, воспользуйтесь ссылкой на событие временной шкалы.  

## Общие свойства событий временной шкалы  

Некоторые сведения представлены в событиях всех типов, в то время как некоторые применимы только к определенным типам событий.  В этом разделе перечислены свойства, которые являются общими для различных типов событий.  Свойства, специфичные для определенных типов событий, перечислены в ссылках на следующие типы событий.  

| Свойство | Когда оно отображается |  
|:--- |:--- |  
| Агрегированное время | Для событий с **вложенными событиями**— время, затраченное на каждую категорию событий. |  
| Стек вызовов | Для событий с **дочерними событиями**— время, затраченное на каждую категорию событий. |  
| Время ЦП | Сколько времени ЦП потратило на записанное событие. |  
| Сведения | Другие сведения о событии. |  
| Duration \ (в заметках времени \) | Сколько времени занимает событие со всеми дочерними элементами; timestamp — время возникновения события, относительно момента начала записи. |  
| Собственное время | Время, в течение которого событие не дочернего объекта. |  
| Размер использованной кучи | Объем памяти, используемой приложением, когда событие было записано, а дельта \ (+/-) изменяется в используемом размере кучи с момента последнего выборки. |

## Загрузка событий  

В этом разделе перечислены события, которые относятся к загрузке категории и их свойствам.  

| Событие | Описание |  
|:--- |:--- |  
| Анализ HTML-кода |  Microsoft Edge запустил алгоритм синтаксического анализа HTML. |  
| Завершить загрузку |  Сетевой запрос выполнен. |  
| Получение данных |  Получены данные для запроса.  Одно или несколько событий приема данных. |  
| Получение ответа |  Первоначальный ответ HTTP из запроса. |  
| Отправить запрос |  Сетевой запрос отправлен. |  

### Загрузка свойств событий  

| Свойство | Описание |  
|:--- |:--- |  
| Ресурс | URL-адрес запрашиваемого ресурса. |  
| Preview | Предварительный просмотр запрашиваемого ресурса \ (только изображения \). |  
| Метод запроса | Метод HTTP, используемый для запроса ( `GET` `POST` например, "\"). |  
| Код состояния | Код ответа HTTP. |  
| Тип MIME | Тип MIME запрашиваемого ресурса. |  
| Длина закодированных данных | Длина запрашиваемого ресурса в байтах. |  

## События сценария  

В этом разделе перечислены события, которые относятся к категории сценариев и их свойствам.  

| Событие | Описание |  
|:--- |:--- |  
| Кадр анимации, срабатывающий | Назначенный кадр анимации, а также его обработчик обратного вызова. |  
| Отмена анимации кадра |  Запланированный кадр анимации был отменен. |  
| Событие GC |  Произошел сбор мусора. |  
| DOMContentLoaded |  [Событие DOMContentLoaded][MDNWindowDOMContentLoadedEvent] было инициировано браузером.  Это событие возникает, когда все содержимое модели DOM страницы загружено и проанализировано. |  
| Оценка сценария | Вычислен сценарий. |  
| Событие | Событие JavaScript \ (например, `mousedown` или `key` \). |  
| Вызов функции | Сделан вызов функции JavaScript верхнего уровня (появляется только в том случае, если браузер входит в ядро JavaScript). |  
| Установка таймера | Таймер создан с помощью [setInterval ()][MDNWindowOrWorkerGlobalScopeSetInterval] или [setTimeout ()][MDNWindowOrWorkerGlobalScopeSetTimeout]. |  
| Кадр анимации запроса | `requestAnimationFrame()`Звонок, запланированный новым кадром. |  
| Удалить таймер | Ранее созданный таймер был очищен. |  
| Время |  Сценарий, именуемый [Console. Time ()][ConsoleApiTime]. |  
| Время окончания | Сценарий, именуемый [Console. timeEnd ()][ConsoleApiTimeEnd]. |  
| Таймер запущен | Таймер, который был запланирован с помощью `setInterval()` или `setTimeout()` . |  
| Изменение состояния XHR Ready | Состояние готовности XMLHTTPRequest изменилось. |  
| Загрузка XHR | `XMLHTTPRequest`Загрузка завершена. |  

### Свойства событий в сценариях  

| Свойство | Описание |  
|:--- |:--- |  
| КОД таймера | Идентификатор таймера. |  
| Превышено время ожидания | Время ожидания, указанное в таймере. |  
| Повторяет | Логическое значение, определяющее, повторяется ли таймер. |  
| Вызов функции | Вызываемая функция. |  

## Обработка событий  

В этом разделе перечислены события, которые относятся к категории отрисовки и их свойствам.  

| Событие | Описание |  
|:--- |:--- |  
| Недействительный макет | Макет страницы был аннулирован при изменении модели DOM. |  
| Макет | Макет страницы завершен. |  
| Пересчитать стиль | Стили элементов, пересчитанные Microsoft Edge. |  
| Scroll | Содержимое вложенного представления было прокручено. |  

### Свойства событий отрисовки  

| Свойство | Описание |  
|:--- |:--- |  
| Макет "недействителен" | Для записей макета трассировка стека кода, который привел к недействительности макета. |  
| Узлы, для которых нужен макет | Для записей макета — количество узлов, которые были помечены как нужные макеты до начала перекомпоновки.  Обычно это узлы, которые были аннулированы кодом разработчика, и путь вверх, чтобы переадресовать корневой макет. |  
| Размер дерева макета | Для записей макета — общее количество узлов в корне перемакета (узел, на котором Microsoft Edge запускает перекомпоновку). |  
| Область макета | Возможные значения: `Partial` \ (граница макета — часть DOM \) или `Whole document` . |  
| Затронутые элементы | Для пересчета записей стилей — количество элементов, затронутых с помощью пересчета стилей. |  
| Недействительные стили | Для пересчета записей стилей — трассировка стека кода, который вызвал недействительность стиля. |  

## События рисования  

В этом разделе перечислены события, которые относятся к категории "раскрасить" и их свойствам.  

| Событие | Описание |  
|:--- |:--- |  
| Составные слои | Составные слои изображений для модуля отрисовки Microsoft Edge. |  
| Декодирование изображений | Перекодировано ресурс изображения. |  
| Изменение размера изображения | Размер изображения был изменен с помощью собственных размеров. |  
| Paint | Составные слои были окрашены в область экрана.  При наведении указателя мыши на запись Paint выделена область экрана, которая была обновлена. |  

### Свойства событий рисования  

| Свойство | Описание |  
|:--- |:--- |  
| Location | Для событий Paint координаты x и y прямоугольника рисования. |  
| Кодов | Для событий рисования — высота и Ширина закрашиваемой области. |
