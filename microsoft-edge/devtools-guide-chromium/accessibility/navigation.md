<span data-ttu-id="16c61-250">Сосредоточив внимание **на** области "Доступность", выберите перемещение фокуса внутри и изучение `Tab` содержимого.</span><span class="sxs-lookup"><span data-stu-id="16c61-250">With focus on the **Accessibility** pane, select `Tab` to move focus inside and explore the contents.</span></span>  <span data-ttu-id="16c61-251">На области ["Доступность"][DevtoolsAccessibilityReference] есть элементы управления для изучения дерева доступности, атрибутов ARIA, примененных к элементу, и вычисляемой свойствах.</span><span class="sxs-lookup"><span data-stu-id="16c61-251">On the [Accessibility pane][DevtoolsAccessibilityReference] there are controls for exploring the accessibility tree, the ARIA attributes applied to an element, and the computed accessibility properties.</span></span>

<span data-ttu-id="16c61-253">Дерево **доступности представлено** в качестве дерева [ARIA,][W3CWaiAriaTree] где каждый из них соответствует `treeitem` элементу в DOM.</span><span class="sxs-lookup"><span data-stu-id="16c61-253">The **Accessibility Tree** is presented as an [ARIA tree][W3CWaiAriaTree] where each `treeitem` corresponds to an element in the DOM.</span></span>  <span data-ttu-id="16c61-254">Дерево объявляет вычисленную роль для выбранного узла.</span><span class="sxs-lookup"><span data-stu-id="16c61-254">The tree announces the computed role for the selected node.</span></span>  <span data-ttu-id="16c61-255">Универсальные элементы, такие как GenericContainer, в дереве объявляются `div` `span` как "GenericContainer".</span><span class="sxs-lookup"><span data-stu-id="16c61-255">Generic elements like `div` and `span` are announced as "GenericContainer" in the tree.</span></span>  <span data-ttu-id="16c61-256">Используйте клавиши со стрелками для обхода дерева и изучения родительских и родительских отношений.</span><span class="sxs-lookup"><span data-stu-id="16c61-256">Use the arrow keys to traverse the tree and explore parent-child relationships.</span></span>
