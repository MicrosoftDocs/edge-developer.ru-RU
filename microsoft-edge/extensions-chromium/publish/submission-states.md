# Состояние отправки расширений и общие сведения  

На странице "Обзор" в центре партнеров отображается состояние расширения в общем потоке отправки.  

| # |  Состояние |  Описание и сведения |  
|:--- |:--- |:--- |  
| 1,1 |  В черновике |  После создания отправки и сохранения ее в учетной записи состояние изменится на это состояние.  <br />  Вы не отправили пакет расширения и сведения о форме отправки для публикации в Microsoft Store.  <br />  Ваше расширение недоступно для пользователей в этом состоянии.  |  
| 2|  На рассмотрении |  После отправки расширения состояние изменится на это состояние.  <br />  Пакет расширения и сведения о форме отправки просматриваются корпорацией Майкрософт.  <br />  Ваше расширение недоступно для пользователей в этом состоянии.  |  
| Трехконтактный|  Ожидание публикации |  После успешного завершения проверки расширения и при подготовке расширения к публикации в Microsoft Store состояние изменится на это состояние.  <br />  Это промежуточное состояние между `In review` и `In the store` .  <br />  Это состояние может не отображаться для всех отправленных данных.  |  
| четырехпроцессорном|  В магазине |  После завершения рецензирования и публикации Вашего расширения в Microsoft Store состояние изменится на это состояние.  <br />  Ваше расширение доступно в магазине Microsoft Store на указанных вами рынках.  |  
| 5 |  В магазине.  Обновление в рецензировании |  Ваше расширение Опубликовано в Microsoft Store, и вы отправили обновление, которое находится в рецензировании корпорацией Microsoft.  |  
| 152 |  Проверка не пройдена |  Если расширение не пройдет проверку, состояние изменится на это состояние.  <br />  В ходе начального рецензирования или при обновлении может возникнуть ошибка проверки.  <br />  Предполагается, что вы хотите сделать корректирующее действие и повторно отправьте расширение для публикации в Microsoft Store.  |  
| 5-7 |  Недоступно в магазине |  Существует три возможных сценария, в которых ваше расширение отображает это состояние: **Неопубликованные из магазина** , **удаленные из магазина**и **заблокированы**.  Описание каждого из этих трех элементов указано ниже.  |  
| No8 |  Неопубликованные из магазина |  Вы отопубликовали свое расширение из Microsoft Store в центре партнеров.  <br />  В центре партнера на странице отправки расширения нажмите кнопку Отменить публикацию.  <br />  После отмены публикации расширение больше не будет доступно в Microsoft Store для новых пользователей, но существующие пользователи могут продолжать использовать свои копии ваших расширений.  |  
| @ |  Удалено из магазина |  Если ваше расширение нарушает условия и положения в Microsoft Store, корпорация Майкрософт может удалить ее из магазина Microsoft Store и изменить состояние на.  <br />  После того как вы разрешаете свое расширение в Майкрософт, ваше расширение больше не доступно в Microsoft Store для загрузки новых пользователей, но существующие пользователи могут продолжать использовать свои копии вашего расширения.  |  
| 5-10 |  Заблокировано |  Если ваше расширение является вредоносным и может порушить безопасность и конфиденциальность пользователей, корпорация Майкрософт имеет право заблокировать ваше расширение из Microsoft Store и изменить состояние на.  <br />  Если расширение заблокировано, расширение удаляется из Microsoft Store, а также удаляется из пользовательских устройств.  |
