# JsPropertyIdRef Typedef

Идентификатор свойства.  
  
## Синтаксис  
  
```cpp  
typedef JsRef JsPropertyIdRef;  
```  
  
## Комментарии  
 Идентификаторы свойств используются для ссылки на свойства объектов JavaScript, а не на строки.  
  
## Требования  
 **Заглавная:** jsrt.h
