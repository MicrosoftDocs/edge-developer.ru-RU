# JsValueRef Typedef

Ссылка на значение JavaScript.  
  
## Синтаксис  
  
```cpp 
typedef JsRef JsValueRef;  
```  
  
## Комментарии  
 Значение JavaScript — это один из следующих типов значений: Undefined, Null, Boolean, String, Number или Object.  
  
## Требования  
 **Заглавная:** jsrt.h
